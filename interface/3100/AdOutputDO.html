AdOutputDO

Description

The AdOutputDO function writes data to the digital output pins on the board.

Syntax

C

INT AdOutputDO(
HANDLE  hDeviceHandle,
DWORD  dwData
);

Visual Basic

Declare Function AdOutputDO Lib "FbiAd.DLL" ( _
ByVal hDeviceHandle As Long, _
ByVal  dwData As Long _

) As Long

Delphi

function AdOutputDO(
hDeviceHandle : THandle;
   dwData   : DWORD

) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
   
dwData Specifies the digital data to be output. Please refer to "Data Format."

Return Value

The AdOutputDO function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

The AdOutputDO function is not applicable to the boards (PCI-3161 and PCI-3163) that have no digital output pins.

To use digital output on the PCI-3525, and CTP/CPZ-3525, change either CN3 or CN4 to digital output pin by the AdSetFunction function.

To use digital output on the PCI-361812, change the pin to be used to digital output pin by the AdSetFunction function.

Examples

C

nRet = AdOutputDO( hDeviceHandle, 0x03 );

Visual Basic

nRet = AdOutputDO( hDeviceHandle, &H03 )

Delphi

nRet := AdOutputDO( hDeviceHandle, $03 );
Output the data 03h to the general purpose digital output pins on the board specified by hDeviceHandle.
