  • The AdStartSampling function, AdStartFileSampling function, AdSyncSampling function, AdBmStartFileSampling  function

    To use the external clock set 0 to the following members.
    Function Structure Member
    AdSetSamplingConfig ADSMPLREQ fSmplFreq
    AdBmSetSamplingConfig ADBMSMPLREQ fSmplFreq
    AdMemSetSamplingConfig ADMEMSMPLREQ fSmplFreq


    To use the external trigger, set AD_EXTTRG or AD_EXTTRG_DI to the following members.

    Function Structure Member
    AdSetSamplingConfig ADSMPLREQ ulTrigMode
    AdBmSetSamplingConfig ADBMSMPLREQ ulStartMode, ulStopMode
    AdMemSetSamplingConfig ADMEMSMPLREQ ulStopMode

For the board/card whose pins used in the external clock and external trigger are both EXTINT IN, the external clock and external trigger cannot be used simultaneously.  

  • The AdMemTriggerSampling function
    To use the external clock, set fSmpFreq (parameter) to 0.
