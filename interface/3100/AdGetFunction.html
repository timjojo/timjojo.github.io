AdGetFunction

Description

The AdGetFunction function retrieves the functional configuration of the CN3 and CN4 connectors on the PCI-3525, CTP/CPZ-3525.

 The AdGetFunction function retrieves the functional configuration of the  connector on the CTP/CPZ-360810 and LPC/PEX-320724.

Syntax

C 

INT AdGetFunction(
HANDLE  hDeviceHandle,
ULONG  ulChNo,
ULONG  *ulFunction
);

Visual Basic

Declare Function AdGetFunction Lib "FbiAd.DLL" (_
ByVal  hDeviceHandle  As Long, _
ByRef  ulChNo  As Long, _
ByRef  ulFunction  As Long _
) As Long

Delphi

function AdGetFunction(
hDeviceHandle  : THandle;
var   ulChNo  : Cardinal;
var  ulFunction  : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
             
ulChNo Specifies the connector number to retrieve the functional configuration. 
Specify 3 or 4 for PCI-3525 or CTP/CPZ-3525.
CTP/CPZ-360810
Code Value Description
AD_EX_DIO1 1 CN4
AD_EX_DIO2 2 CN5
AD_EX_DIO3 3 CN6 10 pin
AD_EX_DIO4 4 CN6 11 pin
AD_EX_DIO5 5 CN6 12 pin
LPC/PEX-320724
Code Value Description
AD_EX_DIO1 1 CN3
                          
ulFunction Points to a variable to receive the function of the connector specified by ulChNo.
PCI-3525、CTP/CPZ-3525
Code Value Description
AD_CN_FREE 0 The connector is not used.
AD_CN_EXTRG_IN 1 External trigger input
AD_CN_EXTRG_OUT 2 External trigger output
AD_CN_EXCLK_IN 3 External clock input
AD_CN_EXCLK_OUT 4 External clock output
AD_CN_EXINT_IN 5 External interrupt input
AD_CN_ATRG_OUT 6 Analog trigger output
AD_CN_DI 7 General purpose digital input
AD_CN_DO 8 General purpose digital output
AD_CN_OPEN 10 Open status
(Only applicable to CN4)
AD_CN_EXSMP1_OUT 12 External sampling status 1 output 
AD_CN_EXSMP2_OUT 13 External sampling status 2 output 
CTP/CPZ-360810
Code Value Description
AD_CN_FREE 0 The connector is not used.
AD_CN_EXTRG_IN 1 External trigger input
AD_CN_EXTRG_OUT 2 External trigger output
AD_CN_EXCLK_IN 3 External clock input
AD_CN_EXCLK_OUT 4 External clock output
AD_CN_EXINT_IN 5 External interrupt input
AD_CN_ATRG_OUT 6 Analog trigger output
LPC/PEX-320724
Code Value Description
AD_CN_FREE 0 The connector is not used.
AD_CN_EXTRG_IN 1 External trigger input
AD_CN_EXCLK_IN 3 External clock input

Return Value

The AdGetFunction function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • This function is applicable to only in the PCI-3525, CTP/CPZ-3525, CTP/CPZ-360810 and  LPC/PEX-320724.
PCI-3525, CTP/CPZ-3525
  • The default setting is retrieved if this function is called before configuring the function by using the AdSetFunction function.
  • The default setting of CN3 is AD_CN_FREE.
  • When a connector setting except for DA_CN_FREE is specified on CN3 of analog output partition, an error returns without retrieving the configuration.
  • The default setting of CN4 is AD_CN_OPEN.
  • While the analog output is performed on CN4 of the analog output partition, an error returns without retrieving the configuration.
CTP/CPZ-360810
  • The default setting is AD_CN_FREE.
LPC/PEX-320724
  • The default setting is AD_CN_FREE

Examples

C 

ULONG ulFunction;
            
nRet = AdGetFunction( hDeviceHandle, 3, &ulFunction );

Visual Basic

Dim ulFunction As Long
          
nRet = AdGetFunction( hDeviceHandle, 3, _ ulFunction )

Delphi

var
    uFunction: Cardinal;
                     
nRet := AdGetFunction( hDeviceHandle, 3, uFunction );
Retrieve the functional configuration of the CN3 connector on the board specified by hDeviceHandle.
