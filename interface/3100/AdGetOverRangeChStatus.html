AdGetOverRangeChStatus

Description

The AdGetOverRangeChStatus function retrieves the event status of automatic range-switching capability at overvoltage input. 

Syntax

C 

INT AdGetOverRangeChStatus(
HANDLE  hDeviceHandle,
ULONG  *ulChStatus
);

Visual Basic

Declare Function AdGetOverRangeChStatus Lib _ "FbiAd.DLL" ( _
ByVal  hDeviceHandle  As Long, _ 
ByRef  ulChStatus As Long _
) As Long

Delphi

function AdGetFunction(
hDeviceHandle  : THandle;
var   ulChStatus : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
             
ulChStatus Points an array to store the range status of each channel.
Code Description
AD_STATUS_OVPM_NORMAL Specified range status
AD_STATUS_OVPM_HIGH_RANGE Automatic range-switching (+/-50 V) or inside GND connection status

Return Value

The AdGetOverRangeChStatus function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Examples

C 

ULONG ulEventStatus[16];
            
nRet = AdGetOverRangeChStatus( hDeviceHandle, ulEventStatus );

Visual Basic

Dim ulEventStatus(0 to 15) As Long
          
nRet = AdGetOverRangeChStatus( hDeviceHandle, _ ulEventStatus(0) )

Delphi

var
    ulEventStatus : array[0..15] of Cardinal;
                     
nRet := AdGetOverRangeChStatus( hDeviceHandle, @ulEventStatus[0] );
Retrieve the status of the current full-scale detection event on the board/card specified by hDeviceHandle.
