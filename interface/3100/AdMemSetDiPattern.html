AdMemSetDiPattern

Description

The AdMemSetDiPattern function configures the analog-to-digital input pattern matching conditions. 

Syntax

C 
INT AdMemSetDiPattern(
HANDLE  hDeviceHandle,
ULONG ulCh,
ULONG *bPatternTrug
);
Visual Basic
Declare Function AdMemSetDiPattern Lib "FbiAd.DLL" ( _
ByVal hDeviceHandle As Long, _
ByVal ulCh As Long, _
ByRef ulPatternTrug As Long _
) As Long
Delphi
function AdMemSetDiPattern(
hDeviceHandle   : THandle;
ulCh : Cardinal;
var ulPatternTrug : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
    
ulCh Specifies the number of channels to perform the DI pattern matching in the range of 1 through 8. Specifies 1 or more larger value.
    
ulPatternTrug Points to a condition setting variable of DI pattern matching trigger. 
   
ulPattern[0]: DI1
ulPattern[1]: DI2
ulPattern[2]: DI3
ulPattern[3]: DI4
ulPattern[4]: DI5
ulPattern[5]: DI6
ulPattern[6]: DI7
ulPattern[7]: DI8
   
Code Description
AD_DISABLE No setting (default setting)
AD_LOW_LEVEL Input signal status: 0 (low level)
AD_HIGH_LEVEL Input signal status: 1 (high level)
AD_DOWN_EDGE  Falling edge
AD_UP_EDGE  Rising edge

Return Value

The AdMemSetDiPattern function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • This function is not available while the sampling is running.
  • This function is enable to use when the sampling end condition is configured to the DI pattern matching （AD_STOP_DI_EQ）by using the AdMemSetSamplingConfig function.
  • This function cannot use with the digital output.
  • This function is applicable to the CSI/CBI-320110, CBI-320110TK, CBI-320110TL, LPC-320910, PEX-320910 and CTP/CPZ-360810.

Examples

C 
ULONG ulPatternTrig[2];
   
ulPatternTrig[0] = AD_LOW_LEVEL; // DI1
ulPatternTrig[1] = AD_UP_EDGE;      // DI2
   
nRet = AdMemSetDiPattern( hDeviceHandle, 2, &ulPatternTrig[0] );
Visual Basic
Dim ulPatternTrig(1) As Long
   
ulPatternTrig(0) = AD_LOW_LEVEL ' DI1
ulPatternTrig(1) = AD_UP_EDGE ' DI2
   
nRet = AdMemSetDiPattern( hDeviceHandle, 2, ulPatternTrig(0) )
Delphi

var

   ulPatternTrig: array[0..1] of Cardinal;

    

   ulPatternTrig[0] := AD_LOW_LEVEL; // DI1

   ulPatternTrig[1] := AD_UP_EDGE;      // DI2

    

nRet := AdMemSetDiPattern( hDeviceHandle, 2, ulPatternTrig(0) );
Specify the digital input pattern matching conditions on the board/card specified by hDeviceHandle as follows.

- DI1: Low level
- DI2: Raising edge
