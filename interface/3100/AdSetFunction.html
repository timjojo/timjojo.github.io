13.5.40 AdSetFunction

Description

The AdSetFunction function configures the functional configuration of the CN3 and CN4 connectors. This function is applicable only to the PCI-3525 and CTP/CPZ-3525.

The AdSetFunction function configures the functional configuration of the connector on the CTP/CPZ-360810 and  LPC/PEX-320724.

Syntax

C 

INT AdSetFunction(
HANDLE  hDeviceHandle,
ULONG  ulChNo,
ULONG  ulFunction
);

Visual Basic

Declare Function AdSetFunction Lib _ "FbiAd.DLL" ( _
ByVal  hDeviceHandle As Long, _
ByVal  ulChNo  As Long, _
ByVal  ulFunction  As Long _
) As Long

Delphi

function AdSetFunction(
hDeviceHandle : THandle;
ulChNo  : Cardinal;
ulFunction  : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
                  
ulChNo Specifies the connector number to configure the function. 
Specify 3 or 4 for the PCI-3525 and CPZ-3525.
        
CTP/CPZ-360810
Code Value Description
AD_EX_DIO1 1 CN4
AD_EX_DIO2 2 CN5
AD_EX_DIO3 3 CN6 10pin
AD_EX_DIO4 4 CN6 11pin
AD_EX_DIO5 5 CN6 12pin
LPC/PEX-320724
Code Value Description
AD_EX_DIO1 1 CN3
ulFunction Specifies the function of the connector specified by ulChNo.
The functions are specified with the following codes. 
PCI-3525 and CPZ-3525
Code Value Description
AD_CN_FREE 0 The connector is not used.
AD_CN_EXTRG_IN 1 External trigger input
AD_CN_EXTRG_OUT 2 External trigger output
AD_CN_EXCLK_IN 3 External clock input
AD_CN_EXCLK_OUT 4 External clock output
AD_CN_EXINT_IN 5 External interrupt input
AD_CN_ATRG_OUT 6 Analog trigger output
AD_CN_DI 7 Digital input
AD_CN_DO 8 Digital output
AD_CN_OPEN 10 Open status 
(Only applicable to CN4)
AD_CN_EXSMP1_OUT 12 External sampling status 1 output
AD_CN_EXSMP2_OUT 13 External sampling status 2 output
CTP/CPZ-360810
Code Value Description
AD_CN_FREE 0 The connector is not used.
AD_CN_EXTRG_IN 1 External trigger input
AD_CN_EXTRG_OUT 2 External trigger output
AD_CN_EXCLK_IN 3 External clock input
AD_CN_EXCLK_OUT 4 External clock output
AD_CN_EXINT_IN 5 External interrupt input
AD_CN_ATRG_OUT 6 Analog trigger output
LPC/PEX-320724
Code Value Description
AD_CN_FREE 0 The connector is not used.
AD_CN_EXTRG_IN 1 External trigger input
AD_CN_EXCLK_IN 3 External clock input

Return Value

The AdSetFunction function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

PCI-3525, CTP/CPZ-3525
  • The following table shows the conditions of digital I/O.
    Connector Code Description
    3 AD_CN_DI Makes CN3 to be IN1.
    AD_CN_DO Makes CN3 to be OUT1.
    4 AD_CN_DI Makes CN4 to be IN2.
    AD_CN_DO Makes CN4 to be OUT2.
  • The same function cannot be specified to both CN3 and CN4.
  • AD_CN_FREE is specified at the default setting on CN3.
  • AD_CN_OPEN is specified at the default setting on CN4.
  • While the digital-to-analog output is performed at the digital-to-analog output partition, do not configure the CN4 setting.
  • The external sampling status 1 output outputs the status from start trigger generation to stop trigger detection.

CTP/CPZ-360810 and LPC/PEX-320724

  • AD_CN_FREE is specified at the default setting.

Examples

C 

nRet = AdSetFunction( hDeviceHandle, 3, AD_CN_EXTRG_IN );

Visual Basic

nRet = AdSetFunction( hDeviceHandle, 3, _ AD_CN_EXTRG_IN )

Delphi

nRet := AdSetFunction( hDeviceHandle, 3, AD_CN_EXTRG_IN );
Configure the  functional configuration of the CN3 connector to an external trigger input on the board specified by hDeviceHandle.
