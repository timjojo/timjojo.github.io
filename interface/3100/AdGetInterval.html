AdGetInterval

Description

The AdGetInterval function retrieves the interval timer cycle. 

Syntax

C 

INT AdGetInterval(
HANDLE  hDeviceHandle,
ULONG  *ulInterval
);

Visual Basic

Declare Function AdGetInterval Lib _ "FbiAd.DLL" ( _
ByVal  hDeviceHandle As Long, _ 
ByRef  ulInterval  As Long _
) As Long

Delphi

function AdGetInterval(
hDeviceHandle  : THandle;
var  ulInterval  : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
                
ulInterval Points to a variable to receive the interval timer cycle in the range of 1 through 16777215 in microseconds. If the cycle is 0, the interval timer is stopped.

Return Value

The AdGetInterval function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Examples

C

ULONG ulInterval;
nRet = AdGetInterval( hDeviceHandle, &ulInterval );

Visual Basic

Dim ulInterval As Long
nRet = AdGetInterval( hDeviceHandle, ulInterval )

Delphi

var ulInterval: Cardinal;
nRet := AdGetInterval( hDeviceHandle, ulInterval );
Retrieve the interval timer cycle on the board specified by hDeviceHandle.
