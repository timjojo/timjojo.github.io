AdSetFilter

Description

The AdSetFilter function configures the digital filter of the board/card.

Syntax

C

INT AdSetFilter(
 HANDLE hDeviceHandle,
  ULONG ulFilter
);

Visual Basic

Declare Function AdSetFilter Lib "FbiAd.DLL" ( _
 ByVal hDeviceHandle As Long, _
 ByVal ulFilter As Long _
) As Long

Delphi

function AdSetFilter(
 hDeviceHandle : THandle;
 ulFilter : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameter
hDeviceHandle Specifies the device handle obtained by the AdOpen function.
 
ulFilter  Specifies either one of digital filter code. 

The digital filter is specified by the following code. 
Code Value Descriptiojn
AD_DF_8 0 Over sampling rate: 8
AD_DF_16 1 Over sampling rate: 16
AD_DF_32 2 Over sampling rate: 32
AD_DF_64 3 Over sampling rate: 64
AD_DF_128 4 Over sampling rate: 128
AD_DF_256 5 Over sampling rate: 256

Return Value

AdSetFilter function returns AD_ERROR_SUCCESS if the process is successfully completed. 
Otherwise, this function returns another code. Please refer to the error codes.

Comments

-This function cannot be performed during sampling.
-Refer to the product manual for more detail about over sampling rate.
-This function is applicable only to the LPC-320724 and PEX-320724.

Example

C

nRet = AdSetFilter( hDeviceHandle, AD_DF_256);

Visual Basic

nRet = AdSetFilter( hDeviceHandle, AD_DF_256)

Delphi

nRet := AdSetFilter( hDeviceHandle, AD_DF_256);

Sets the over sampling rate to 256 on the board/card specified by hDeviceHandle.
