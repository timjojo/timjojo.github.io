AdSetInterval

Description

The AdSetInterval function configures the interval timer cycle.

Syntax

C 

INT AdSetInterval(
HANDLE  hDeviceHandle,
ULONG  ulInterval
);

Visual Basic

Declare Function AdSetInterval Lib _ "FbiAd.DLL" ( _
ByVal  hDeviceHandle As Long, _
ByVal  ulInterval  As Long _
) As Long

Delphi

function AdSetInterval(
hDeviceHandle  : THandle;
ulInterval  : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
                
ulInterval Specifies the interval timer cycle in the range of 0 through 16777215 in µs. If the cycle is 0, the interval timer will be stopped.

Return Value

The AdSetInterval function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

  • The interval timer cycle specified by the AdSetInterval function is used when  the sampling start condition is set to the interval timer cycle.
  • When the sampling start condition is set to the interval timer cycle, each sampling frame starts every specified interval timer cycle.
  • In sampling start condition, samplings starts after AdStartSampling function is executed and reaches the specified cycle.  
  • In sampling stop condition, samplings stops after AdStartSampling function is executed and reaches the specified cycle.  

Examples

C 

nRet = AdSetInterval( hDeviceHandle, 1000 );

Visual Basic

nRet = AdSetInterval( hDeviceHandle, 1000 )

Delphi

nRet := AdSetInterval( hDeviceHandle, 1000 );
Configure the interval timer cycle to 1 ms on the board/card specified by hDeviceHandle.
