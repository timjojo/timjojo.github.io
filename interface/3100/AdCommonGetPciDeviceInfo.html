AdCommonGetPciDeviceInfo

Description

The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).

DWORD  *dwBaseAddress5,
DWORD  *dwSubsystemID,
DWORD  *dwSubsystemVendorID, 
DWORD  *dwInterruptLine, 
DWORD  *dwBoardID
);

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
                
dwDeviceID  Points to a variable to receive the device ID. Use dwSubsystemID to distinguish PCI boards from CompactPCI boards, and vice versa. 
                
Example)
3133 (0C3Dh in hexadecimal) is for the PCI-3133 and CTP-3133
                                         
dwVendorID  Points to a variable the vendor ID. The vendor ID of Interface Corporation is 1147h.
                         
dwClassCode Points to a variable to receive the class code of the device.
                      
dwRevisionID Points to a variable to receive the revision ID.
                 
dwBaseAddress0 Points to a variable to receive the base address register 0.
            
dwBaseAddress1 Points to a variable to receive the base address register 1.
                  
dwBaseAddress2 Points to a variable to receive the base address register 2.
             
dwBaseAddress3 Points to a variable to receive the base address register 3.
             
dwBaseAddress4 Points to a variable to receive the base address register 4.
               
dwBaseAddress5 Points to a variable to receive the base address register 5.
               
dwSubsystemID Points to a variable to receive the subsystem ID of the board. Bit8 through bit11 in this member identify a type of the product.

Bit8 through Bit11

 Value Type
0h PCI board
1h CompactPCI board
2h CardBus card
                          
dwSubsystemVendorID Points to a variable to receive the subsystem vendor ID of the board.
                     
dwInterruptLine Points to a variable to receive the interrupt line of the board.
                     

dwBoardID

 Points to a variable to receive the board ID of the board.

Return Value

The AdCommonGetPciDeviceInfo function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.
