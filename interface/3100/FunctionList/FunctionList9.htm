List of DLL Functions

PCI-3153

No

Function

Description
1 AdOpen The AdOpen function opens an analog-to-digital input interface module and enables to access to the interface module.
2 AdClose The AdClose function closes the analog-to-digital input interface module and releases the resources. Any subsequent accesses to the interface module are forbidden.
3 AdGetDeviceInfo The AdGetDeviceInfo function retrieves specifications of the interface module.
4 AdSetBoardConfig The AdSetBoardConfig function configures event handling of the interface module.
5 AdGetBoardConfig The AdGetBoardConfig function retrieves an event source called on the interface module.
6 AdSetSamplingConfig The AdSetSamplingConfig function configures sampling conditions of the interface module.
7 AdGetSamplingConfig The AdGetSamplingConfig function retrieves sampling conditions on the interface module.
8 AdAllocateSamplingBuffer The AdAllocateSamplingBuffer function allocates a sampling buffer.
9 AdGetSamplingData The AdGetSamplingData function retrieves sampling data from the interface module.
10 AdFifoGetSamplingData The AdFifoGetSamplingData function retrieves sampling data and digital data from the interface module.
11 AdClearSamplingData The AdClearSamplingData function clears the sampling data in the sampling buffer..
12 AdReadSamplingBuffer The AdReadSamplingBuffer function reads sampling data contained in the sampling buffer.
13 AdStartSampling The AdStartSampling function starts continuous sampling on the interface module.
14 AdStartFileSampling The AdBmStartFileSampling function starts continuous sampling and saves the data into a data file simultaneously.
15 AdTriggerSampling The AdTriggerSampling function samples a single channel on the interface module at each trigger asserted on the EXTRG IN pin.
16 AdMemTriggerSampling The AdMemTriggerSampling function acquires the specified number of samples at each external trigger assertion.
17 AdStopSampling The AdStopSampling function stops continuous sampling performed as an overlapped operation.
18 AdGetStatus The AdGetStatus function retrieves the sampling status on the interface module.
19 AdInputAD The AdInputAD function performs one-shot sampling.
20 AdInputDI The AdInputDI function reads data from the  digital input pins on the interface module.
21 AdOutputDO The AdOutputDO function writes data to the  digital output pins on the interface module.
22 AdAdjustVR The AdReadAdjustVR function loads the electric volume value.
23 AdDataConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted.
24 AdReadFile The AdReadFile function reads the sampling data from a file. 
25 fnConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted.
26 CallBackProc The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed.
27 AdCommonGetPciDeviceInfo The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).
