List of DLL Functions

CTP/CPZ-360810

No Function

Description
1 AdOpen The AdOpen function opens an analog-to-digital input board/card and enables to access to the board/card.
2 AdClose The AdClose function closes the analog-to-digital input board/card and releases the resources. Any subsequent accesses to the board/card are forbidden.
3 AdGetDeviceInfo The AdGetDeviceInfo function retrieves specifications of the board/card.
4 AdSetBoardConfig The AdSetBoardConfig function configures event handling of the interface module.
5 AdGetBoardConfig The AdGetBoardConfig function retrieves an event source called on the board/card.
6 AdMemSetSamplingConfig The AdMemSetSamplingConfig function configures the sampling conditions. 
7 AdMemGetSamplingConfig The AdMemGetSamplingConfig function retrieves the configured sampling conditions. 
8 AdGetSamplingData The AdGetSamplingData function retrieves sampling data from the interface module.
9 AdClearSamplingData The AdClearSamplingData function clears the sampling data in the sampling buffer.
10 AdStartSampling The AdStartSampling function starts continuous sampling on the interface module.
11 AdSyncSampling The AdSyncSampling function enables you to achieve a synchronous sampling on boards/cards connected in parallel.  
12 AdStopSampling The AdStopSampling function stops continuous sampling performed as an overlapped operation.
13 AdGetStatus The AdSyncSampling function enables you to achieve a synchronous sampling on boards/cards connected in parallel.
14 AdInputDI The AdInputDI function reads data from the  digital input pins on the board/card.
15 AdOutputDO The AdOutputDO function writes data to the  digital output pins on the board/card.
16 AdSetRangeEvent The AdSetRangeEvent function configures the event mask setting for the full-scale range detection and automatic range-switching capability at overvoltage input.
17 AdGetRangeEventStatus The AdGetRangeEventStatus function retrieves the status of the full-scale range detection event. 
18 AdResetRangeEvent The AdResetRangeEvent function resets the full-scale range detection event.
19 AdSetOutMode The AdSetOutMode function controls the external output signals. 
20 AdGetOutMode The AdGetOutMode function retrieves the control setting of the external output signals.
21 AdSetFunction The AdSetFunction function configures the functional configuration of the connector on the CTP/CPZ-360810.
22 AdGetFunction The AdGetFunction function retrieves the functional configuration of the  connector on the CTP/CPZ-360810.
23 AdMemSetDiPattern The AdMemSetDiPattern function configures the analog-to-digital input pattern matching conditions. 
24 AdMemGetDiPattern The AdMemGetDiPattern function retrieves the analog-to-digital input pattern matching conditions. 
25 AdDataConv The AdDataConv function converts the form of the analog data. Averaging and interpolation can be done with the conversion.
26 AdReadFile The AdReadFile function reads the sampling data from a file. 
27 fnConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted.
28 CallBackProc The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed.
29 AdCommonGetPciDeviceInfo The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).
