List of DLL Functions

LPC-321012, PEX-321012, PCI-361812

No Function

Description
1 AdOpen The AdOpen function opens an analog-to-digital input board/card and enables to access to the board/card.
2 AdClose The AdClose function closes the analog-to-digital input board/card and releases the resources. Any subsequent accesses to the board/card are forbidden.
3 AdGetDeviceInfo The AdGetDeviceInfo function retrieves specifications of the board/card.
4 AdSetBoardConfig The AdSetBoardConfig function configures event handling of the board/card.
5 AdGetBoardConfig The AdGetBoardConfig function retrieves an event source called on the board/card.
6 AdSetSamplingConfig The AdSetSamplingConfig function configures sampling conditions of the board/card.
7 AdGetSamplingConfig The AdGetSamplingConfig function retrieves sampling conditions on the board/card.
8 AdBmSetSamplingConfig The AdBmSetSamlingConfig function configures sampling conditions.
9 AdBmGetSamplingConfig The AdBmGetSamplingConfig function retrieves sampling conditions.
10 AdAllocateSamplingBuffer The AdAllocateSamplingBuffer function allocates a sampling buffer.
11 AdGetSamplingData The AdGetSamplingData function retrieves sampling data from the board/card.
12 AdClearSamplingData The AdClearSamplingData function clears the sampling data in the sampling buffer.
13 AdReadSamplingBuffer The AdReadSamplingBuffer function reads sampling data contained in the sampling buffer.
14 AdStartSampling The AdStartSampling function starts continuous sampling on the board/card.
15 AdBmStartSampling The AdBmStartSampling function starts continuous sampling on the board/card.(DMA direct transfer)
16 AdStartFileSampling The AdBmStartFileSampling function starts continuous sampling and saves the data into a data file simultaneously.
17 AdStopSampling The AdStopSampling function stops continuous sampling performed as an overlapped operation.
18 AdGetStatus The AdSyncSampling function enables you to achieve a synchronous sampling on boards/cards connected in parallel.
19 AdInputAD The AdInputAD function performs one-shot sampling.
20 AdInputDI The AdInputDI function reads data from the  digital input pins on the board/card.
21 AdOutputDO The AdOutputDO function writes data to the  digital output pins on the board/card.
22 AdSetInterval The AdSetInterval function configures the interval timer cycle.
23 AdGetInterval The AdGetInterval function retrieves the interval timer cycle. 
24 AdSetOutMode The AdSetOutMode function controls the external output signals. 
25 AdGetOutMode The AdGetOutMode function retrieves the control setting of the external output signals.
26 AdSetFunction The AdSetFunction function configures the functional configuration of the control signal of the PCI-361812.
27 AdGetFunction The AdGetFunction function retrieves the functional configuration of the control signal of the PCI-361812.
28 AdAdjustVR The AdAdjustVR function controls the digitally controlled potentiometer on the board/card for calibration purpose.
29 AdReadAdjustVR The AdReadAdjustVR function loads the electric volume value.
30 AdDataConv The AdDataConv function converts the form of the analog data. Averaging and interpolation can be done with the conversion.
31 AdReadFile The AdReadFile function reads the sampling data from a file. 
32 fnConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted.
33 CallBackProc The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed.
34 AdCommonGetPciDeviceInfo The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).
