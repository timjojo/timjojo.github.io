List of DLL Functions

PCI-3161, CTP/CPZ-3120A, CTP/CPZ-3120B, CTP/CPZ-3120C, CTP/CPZ-3126, CTP/CPZ-3133, CTP/CPZ-3135, CTP/CPZ-3165, CTP/CPZ-3168, CTP/CPZ-3170, CTP/CPZ-3171, CTP/CPZ-3172, CTP/CPZ-3173, CTP/CPZ-3177, CTP/CPZ-3178, CTP/CPZ-3180A, CTP/CPZ-3180B, CTP/CPZ-3180C, CTP/CPZ-3521, CTP/CPZ-3522, CTP/CPZ-3523

No Function

Desctiption
1 AdOpen The AdOpen function opens an analog-to-digital input interface module and enables to access to the interface module.
2 AdClose The AdClose function closes the analog-to-digital input interface module and releases the resources. Any subsequent accesses to the interface module are forbidden.
3 AdGetDeviceInfo The AdGetDeviceInfo function retrieves specifications of the interface module.
4 AdSetBoardConfig The AdSetBoardConfig function configures event handling of the interface module.
5 AdGetBoardConfig The AdGetBoardConfig function retrieves an event source called on the interface module.
6 AdSetSamplingConfig The AdSetSamplingConfig function configures sampling conditions of the interface module.
7 AdGetSamplingConfig The AdGetSamplingConfig function retrieves sampling conditions on the interface module.
8 AdAllocateSamplingBuffer The AdAllocateSamplingBuffer function allocates a sampling buffer.
9 AdGetSamplingData The AdGetSamplingData function retrieves sampling data from the interface module.
10 AdClearSamplingData The AdClearSamplingData function clears the sampling data in the sampling buffer.
11 AdReadSamplingBuffer The AdReadSamplingBuffer function reads sampling data contained in the sampling buffer.
12 AdStartSampling The AdStartSampling function starts continuous sampling on the interface module.
14 AdStartFileSampling The AdStartFileSampling function starts continuous sampling and saves the data into a data file simultaneously.
15 AdTriggerSampling The AdTriggerSampling function samples a single channel on the interface module at each trigger asserted on 
the EXTRG IN pin.
16 AdMemTriggerSampling The AdMemTriggerSampling function acquires the specified number of samples
17 AdSyncSampling The AdSyncSampling function enables you to achieve a synchronous sampling on interface modules connected in parallel.
18 AdStopSampling The AdStopSampling function stops continuous sampling performed as an overlapped operation.
19 AdGetStatus The AdGetStatus function retrieves the sampling status on the interface module.
20 AdInputAD The AdInputAD function performs one-shot sampling.
21 AdInputDI The AdInputDI function reads data from the  digital input pins on the interface module.
22 AdOutputDO The AdOutputDO function writes data to the  digital output pins on the interface module.
23 AdSetRangeEvent The AdSetRangeEvent function configures the event mask setting for the full-scale range detection and automatic range-switching capability at overvoltage input.
24 AdGetRangeEventStatus The AdGetRangeEventStatus function retrieves the status of the full-scale range detection event. 
25 AdResetRangeEvent The AdResetRangeEvent function resets the full-scale range detection event.
26 AdAdjustVR The AdAdjustVR function controls the digitally controlled potentiometer on the interface module for calibration purpose.
27 AdDataConv The AdDataConv function converts the form of the analog data. Averaging and interpolation can be done with the conversion.
28 AdReadFile The AdReadFile function reads the sampling data from a file. 
29 fnConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted. 
30 CallBackProc The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed.
31 AdCommonGetPciDeviceInfo The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).
