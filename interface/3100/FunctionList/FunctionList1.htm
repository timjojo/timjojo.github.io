List of DLL Functions

PCI-3120, PCI-3126, PCI-3133, PCI-3135, PCI-3155, PCI-3165, PCI-3166, PCI-3168C, PCI-3170A, PCI-3171A, PCI-3172A, PCI-3173A, PCI-3177C, PCI-3178, PCI-3180, PCI-3521, PCI-3522A, PCI-3523A

No

Function

Description
1 AdOpen The AdOpen function opens an analog-to-digital input board/card and enables to access to the board/card.
2 AdClose The AdClose function closes the analog-to-digital input board/card and releases the resources. Any subsequent accesses to the board/card are forbidden.
3 AdGetDeviceInfo The AdGetDeviceInfo function retrieves specifications of the board/card.
4 AdSetBoardConfig The AdSetBoardConfig function configures event handling of the board/card.
5 AdGetBoardConfig The AdGetBoardConfig function retrieves an event source called on the board/card.
6 AdSetSamplingConfig The AdSetSamplingConfig function configures sampling conditions of the board/card.
7 AdGetSamplingConfig The AdGetSamplingConfig function retrieves sampling conditions on the board/card.
8 AdAllocateSamplingBuffer The AdAllocateSamplingBuffer function allocates a sampling buffer.
9 AdGetSamplingData The AdGetSamplingData function retrieves sampling data from the board/card.
10 AdClearSamplingData The AdClearSamplingData function clears the sampling data in the sampling buffer.
11 AdReadSamplingBuffer The AdReadSamplingBuffer function reads sampling data contained in the sampling buffer.
12 AdStartSampling The AdStartSampling function starts continuous sampling on the board/card.
13 AdSyncSampling The AdSyncSampling function enables you to achieve a synchronous sampling on boards/cards connected in parallel.
14 AdStartFileSampling The AdStartFileSampling function starts continuous sampling and saves the data into a data file simultaneously.
15 AdTriggerSampling The AdTriggerSampling function samples a single channel on the board/card at each trigger asserted on 
the EXTRG IN pin.
16 AdMemTriggerSampling The AdMemTriggerSampling function acquires the specified number of samples at each external trigger assertion.
17 AdStopSampling The AdStopSampling function stops continuous sampling performed as an overlapped operation.
18 AdGetStatus The AdGetStatus function retrieves the sampling status on the board/card.
19 AdInputAD The AdInputAD function performs one-shot sampling.
20 AdInputDI The AdInputDI function reads data from the  digital input pins on the board/card.
21 AdOutputDO The AdOutputDO function writes data to the  digital output pins on the board/card.
22 AdAdjustVR The AdAdjustVR function controls the digitally controlled potentiometer on the board/card for calibration purpose.
23 AdDataConv The AdDataConv function converts the form of the analog data. Averaging and interpolation can be done with the conversion.
24 AdReadFile The AdReadFile function reads the sampling data from a file. 
25 fnConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted. 
26 CallBackProc The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed.
27 AdCommonGetPciDeviceInfo The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).
