List of DLL Functions

PCI-3179,CTP/CPZ-3179

No

Function

Description
1 AdOpen The AdOpen function opens an analog-to-digital input interface module and enables to access to the interface module.
2 AdClose The AdClose function closes the analog-to-digital input interface module and releases the resources. Any subsequent accesses to the interface module are forbidden.
3 AdGetDeviceInfo The AdGetDeviceInfo function retrieves specifications of the interface module.
4 AdSetBoardConfig The AdSetBoardConfig function configures event handling of the interface module.
5 AdGetBoardConfig The AdGetBoardConfig function retrieves an event source called on the interface module.
6 AdSetSamplingConfig The AdSetSamplingConfig function configures sampling conditions of the interface module.
7 AdGetSamplingConfig The AdGetSamplingConfig function retrieves sampling conditions on the interface module.
8 AdLvSetSamplingConfig The AdLvSetSamplingConfig function configures sampling conditions of the interface module. 
9 AdLvGetSamplingConfig The AdLvGetSamplingConfig function retrieves sampling conditions of the interface module.
10 AdAllocateSamplingBuffer The AdAllocateSamplingBuffer function allocates a sampling buffer.
11 AdLvCalibration The AdLvCalibration function runs automatic calibration of the interface module.
12 AdGetSamplingData The AdGetSamplingData function retrieves sampling data from the interface module.
13 AdLvGetSamplingData The AdLvGetSamplingData function retrieves sampling data from the interface module.
14 AdClearSamplingData The AdClearSamplingData function clears the sampling data in the sampling buffer.
15 AdReadSamplingBuffer The AdReadSamplingBuffer function reads sampling data contained in the sampling buffer.
16 AdStartSampling The AdStartSampling function starts continuous sampling on the interface module.
17 AdLvStartSampling The AdLvStartSampling function starts continuous sampling on the interface module.
18 AdStartFileSampling The AdBmStartFileSampling function starts continuous sampling and saves the data into a data file simultaneously.
19 AdStopSampling The AdStopSampling function stops continuous sampling performed as an overlapped operation.
20 AdLvStopSampling

The AdLvStopSampling function stops continuous sampling performed as an overlapped operation.
21 AdGetStatus The AdGetStatus function retrieves the sampling status on the interface module.
22 AdLvGetStatus

The AdLvGetStatus function retrieves the sampling status on the interface module.
23 AdInputAD The AdInputAD function performs one-shot sampling.
24 AdInputDI The AdInputDI function reads data from the  digital input pins on the interface module.
25 AdOutputDO The AdOutputDO function writes data to the  digital output pins on the interface module.
26 AdMeasureTemperature

The AdMeasureTemperature function reads temperature of the I/O connector on the interface moudule.
27 AdDataConv The AdDataConv function converts the form of the analog data. Averaging and interpolation can be done with the conversion. 
28 AdReadFile The AdReadFile function reads the sampling data from a file. 
29 fnConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted.
30 CallBackProc The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed.
31 AdCommonGetPciDeviceInfo The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).
