List of DLL Functions

PCI-3174, PCI-320412, PCI-320416, PCI-360116, PCI-360112, PCI-360216
CTP/CPZ-3174, CTP/CPZ-3175, CTP/CPZ-3182, CTP/CPZ-320412, CTP/CPZ-320416, CTP/CPZ-360116, CTP/CPZ-360112

No

Function

Description
1 AdOpen The AdOpen function opens an analog-to-digital input product and enables to access to the product.
2 AdClose The AdClose function closes the analog-to-digital input product and releases the resources. Any subsequent accesses to the product are forbidden.
3 AdGetDeviceInfo The AdGetDeviceInfo function retrieves specifications of the interface module.
4 AdSetBoardConfig The AdSetBoardConfig function configures event handling of the interface module.
5 AdGetBoardConfig The AdGetBoardConfig function retrieves an event source called on the board/card.
6 AdSetSamplingConfig The AdSetSamplingConfig function configures sampling conditions of the board/card.
7 AdGetSamplingConfig The AdGetSamplingConfig function retrieves sampling conditions on the board/card.
8 AdBmSetSamplingConfig The AdBmSetSamlingConfig function configures sampling conditions.
9 AdBmGetSamplingConfig The AdBmGetSamplingConfig function retrieves sampling conditions.
10 AdAllocateSamplingBuffer The AdAllocateSamplingBuffer function allocates a sampling buffer.
11 AdGetSamplingData The AdGetSamplingData function retrieves sampling data from the interface module.
12 AdBmGetSamplingData The AdBmGetSamplingConfig function retrieves sampling conditions.
13 AdClearSamplingData The AdClearSamplingData function clears the sampling data in the sampling buffer.
14 AdReadSamplingBuffer The AdReadSamplingBuffer function reads sampling data contained in the sampling buffer.
15 AdStartSampling The AdStartSampling function starts continuous sampling on the board/card.
16 AdBmStartSampling The AdBmStartSampling function starts continuous sampling on the interface module.(DMA direct transfer)
17 AdStartFileSampling The AdBmStartFileSampling function starts continuous sampling and saves the data into a data file simultaneously.
18 AdBmStartFileSampling The AdBmStartFileSampling function starts continuous sampling and saves the data into a data file simultaneously.
19 AdSyncSampling The AdSyncSampling function enables you to achieve a synchronous sampling on boards/cards connected in parallel.
20 AdStopSampling The AdStopSampling function stops continuous sampling performed as an overlapped operation.
21 AdGetStatus The AdGetStatus function retrieves the sampling status on the board/card.
22 AdInputAD The AdInputAD function performs one-shot sampling.
23 AdInputDI The AdInputDI function reads data from the  digital input pins on the interface module.
24 AdOutputDO The AdOutputDO function writes data to the  digital output pins on the interface module.
25 AdSetRangeEvent The AdSetRangeEvent function configures the event mask setting for the full-scale range detection and automatic range-switching capability at overvoltage input.
26 AdGetRangeEventStatus The AdGetRangeEventStatus function retrieves the status of the full-scale range detection event. 
27 AdResetRangeEvent The AdResetRangeEvent function resets the full-scale range detection event.
28 AdAdjustVR The AdAdjustVR function controls the digitally controlled potentiometer on the board/card for calibration purpose.
29 AdReadAdjustVR The AdReadAdjustVR function loads the electric volume value.
30 AdDataConv The AdDataConv function converts the form of the analog data. Averaging and interpolation can be done with the conversion.
31 AdReadFile The AdReadFile function reads the sampling data from a file. 
32 fnConv The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted.
33 CallBackProc The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed.
34 AdCommonGetPciDeviceInfo The AdCommonGetPciDeviceInfo function retrieves the configuration registers and the board ID (RSW1).
