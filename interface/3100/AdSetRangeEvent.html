AdSetRangeEvent

Description

The AdSetRangeEvent function configures the event mask setting for the full-scale range detection and automatic range-switching capability at overvoltage input.

Syntax

C 

INT AdSetRangeEvent(
HANDLE  hDeviceHandle,
DWORD  dwEventMask,
DWORD  dwStopMode
);

Visual Basic

Declare Function AdSetRangeEvent Lib _ "FbiAd.DLL" ( _
ByVal  hDeviceHandle As Long, _
ByVal  dwEventMask As Long, _
ByVal  dwStopMode As Long _
) As Long

Delphi

function AdSetRangeEvent(
hDeviceHandle  : THandle;
dwEventMask : DWORD;
dwStopMode : DWORD
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
                
dwEventMask  Specifies the event mask for the full-scale range detection and automatic range-switching capability at overvoltage input.
bit7 bit6 bit5 through bit3 bit2 bit1 bit0
PM2 PM1 0 OV PFE NFE
bit6, bit7 Configures automatic range-switching capability at overvoltage input.
 0: Mask
1: Automatic range-switching (+/-50 V) 
2: Inside GND connection status
bit3 through bit5 Reserved
bit2 Specifies the mask setting of automatic range-switching capability at overvoltage input detection event.
0: Mask
1: Unmask 
bit1 Specifies the mask condition at the positive full-scale range detection event.
0: Mask
1: Unmask 
bit0 Specifies the mask condition at the negative full-scale range detection event.
0: Mask
1: Unmask 
                             
dwStopMode  Specifies the sampling conditions at the full-scale range detection and automatic range-switching capability at overvoltage input.
bit31 through bit3 bit2 bit1 bit0
Reserved OV PFS NFS
bit31 through bit3 Reserved
bit2 Specifies the sampling conditions of automatic range-switching capability at overvoltage input detection.
0: Not stop the sampling.
1: Stop the sampling.
bit1 Specifies the sampling conditions at the positive full-scale range detection.
0: Not stopped
1: Stopped
bit0 Specifies the sampling conditions at the negative full-scale range detection.
0: Not stopped
1: Stopped

Return Value

The AdSetRangeEvent function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • Configure the event handling by the AdSetRangeEvent and AdSetBoardConfig functions to occur an event.
  • The AdGetBoardConfig function retrieves the event regarding to the full-scale range detection and automatic range-switching capability at overvoltage input detection.
  • The AdGetRangeEventStatus function retrieves the status of the full-scale range detection.
  • The AdGetOverRangeChStatus function retrieves the status of the automatic range-switching capability at overvoltage input detection.
  • Refer to "List of Capabilities for Each Model," for more details of the product models and capabilities.

Target Products

Refer to "Corresponding Identifiers for Each Model". 

Examples

C 

nRet = AdSetRangeEvent( hDeviceHandle, 0x0001, 0x0002 );

Visual Basic

nRet = AdSetRangeEvent( hDeviceHandle, H0001, _ H0002 )

Delphi

nRet := AdSetRangeEvent( hDeviceHandle, $0001, $0002 );
Specifies the operation at the full-scale range detection on the board/card specified by hDeviceHandle.
