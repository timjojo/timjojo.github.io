AdOutputSync

Description

The AdOutputSync function configures internal synchronization control of analog input devices.

Syntax

C 

INT AdOutputSync(
   HANDLE  hDeviceHandle,
ULONG   TrgLine,
ULONG  TrgMode
);

Visual Basic 

Declare Function AdOutputSync Lib "FbiAd.DLL" (
   ByVal  hDeviceHandle  As Long, _
ByVal  TrgLine As Long,_
ByVal TrgMode  As Long _
) As Long      

Delphi

function AdOutputSync(
   hDeviceHandle : THandle;
TrgLine: : Cardinal;
TrgMode: : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
 
TrgLine Specifies the signal to perform internal synchronization output. 
Selectable identifiers are as follows. 

Identifier

Value

Description
AD_SYNC_NUM_1  0x0100  Internal synchronization line 1
AD_SYNC_NUM_2  0x0200 Internal synchronization line 2
 
TrgMode Specifies internal synchronization trigger output condition from the following identifiers.

     Identifier

Value

Description
AD_DISABLE  0x80000000 No external trigger output (default)
AD_ATRIG_MODE 8  Trigger timing

Return Value

The AdOutputSync function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Examples

C 

nRet = AdOutputSync( hDeviceHandle, AD_SYNC_NUM_1, AD_ATRIG_MODE );

Visual Basic 

nRet = AdOutputSync( hDeviceHandle, AD_SYNC_NUM_1, AD_ATRIG_MODE )

Delphi

nRet := AdOutputSync( hDeviceHandle, AD_SYNC_NUM_1, AD_ATRIG_MODE );
Performing analog trigger output from the analog-to-digital input device, internal synchronization trigger1 specified in  hDevicehandle.
