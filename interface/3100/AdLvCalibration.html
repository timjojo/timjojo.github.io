AdLvCalibration

Description

The AdLvCalibration function runs automatic calibration of the board/card.

Syntax

C 

INT AdLvCalibration(
   HANDLE  hDeviceHandle,
ULONG  ulChNo,
ULONG  ulCalibration
);

Visual Basic 

Declare Function AdLvCalibration Lib _ "FbiAd.DLL" ( _
   ByVal  hDeviceHandle As Long, _
ByVal  ulChNo As Long, _
ByVal  ulCalibration As Long _
) As Long       

Delphi

function AdLvCalibration(
   hDeviceHandle:  THandle;
ulChNo:  DWORD;
ulCalibration: DWORD
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
    
ulChNo Specifies an analog input channel. Only one channel can be specified.
    
ulCalibration Specifies a calibration method.

Code

Description
AD_SELF_CALIBRATION Self-calibration of the AD converter
AD_ZEROSCALE_CALIBRATION Zero voltage calibration
(system calibration)
AD_FULLSCALE_CALIBRATION Full-scale voltage calibration 
(system calibration)

Return Value

The AdLvCalibration function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • The board/card has to be calibrated by using this function, when you change the range or the sampling rate by using the AdLvSetSamplingConfig function.
  • If you calibrate a channel with the system calibration methods, first, calibrate the channel at the zero voltage, and then calibrate it at the full-scale voltage.
  • Make sure that the correct voltage is applied to the channel according to the calibration method before calling this function.
  • If you change the range and the sampling rate by using the AdSetSamplingConfig function, the self-calibration is executed in the AdSetSamplingConfig function.

Examples

C 

<Self-calibration>
nRet = AdLvCalibration( hDeviceHandle, 1, AD_SELF_CALIBRATION );
<Zero voltage calibration (system calibration)>
nRet = AdLvCalibration( hDeviceHandle, 1, AD_ZEROSCALE_CALIBRATION);
   
<Full-scale voltage calibration (system calibration)>
nRet = AdLvCalibration( hDeviceHandle, 1, AD_FULLSCALE_CALIBRATION);

Visual Basic

nRet = AdLvCalibration( hDeviceHandle, 1, _ AD_SELF_CALIBRATION )
   
<Zero voltage calibration (system calibration)>

Delphi

<Zero voltage calibration (system calibration)>

<Full-scale voltage calibration (system calibration)>
