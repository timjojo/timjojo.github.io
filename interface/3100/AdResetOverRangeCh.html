AdResetOverRangeCh

Description

The AdResetOverRangeChfunction resets the event status of automatic range-switching capability at overvoltage input. 

Syntax

C 

INT AdResetOverRangeCh(
   HANDLE   hDeviceHandle  
);

Visual Basic 

Declare Function AdResetOverRangeCh Lib _ "FbiAd.DLL" ( _
   ByVal   hDeviceHandle As Long _
) As Long      

Delphi

function AdResetOverRangeCh(
   hDeviceHandle : THandle  
):DWORD; stdcall; external 'FbiAd.DLL';

Parameter

hDeviceHandle  Specifies the device handle obtained by the AdOpen function.

Return Value

The AdResetOverRangeCh function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

After occurring an event of automatic range-switching capability at overvoltage input, the next event will not occur. Because this function is waiting for the event reset completion.

Examples

C 

nRet = AdResetOverRangeCh( hDeviceHandle );

Visual Basic 

nRet = AdResetOverRangeCh( hDeviceHandle )

Delphi

nRet := AdResetOverRangeCh( hDeviceHandle );
Reset the event status of automatic range-switching capability at overvoltage input of the board/card specified by hDeviceHandle.
