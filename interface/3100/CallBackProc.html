CallBackProc

Description

The CallBackProc function is a placeholder for a callback routine. This function is called when the sampling is completed. Supply a pointer to your function for the lpEventProc parameter in the AdSetBoardConfig function.

Syntax

C
void CALLBACK CallBackProc(
DWORD  dwUser    
);
Visual Basic
Sub CallBackProc( _
ByVal  dwUser As Long _  
)
Delphi
procedure CallBackProc(
dwUser  : DWORD    
); stdcall;

Parameter

dwUser Contains the data passed to dwUser in the AdSetBoardConfig function.

Return Value

This function has no return value.
