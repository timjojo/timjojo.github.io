11.5 Return Values
     

Error Code

Description

Comment/Solution

AD_ERROR_SUCCESS The process was successfully completed.

  -
AD_ERROR_NOT_DEVICE The specified driver cannot be called. The specified device is not found. Make sure that the board/card and the device driver correctly installed in your computer.
AD_ERROR_NOT_OPEN The specified driver cannot be opened. Errors occurred while the system opens the device.
AD_ERROR_INVALID_HANDLE The device handle is invalid. Use the device handle obtained by the AdOpen function.
AD_ERROR_ALREADY_OPEN The specified device cannot be opened because it has already been opened by another process. The device used by another process cannot be accessed.
AD_ERROR_NOT_SUPPORTED The specified capability is not supported. The specified capability cannot be used.
AD_ERROR_NOW_SAMPLING The sampling is running now. The specified sampling has already been called. The specified function is not available while the sampling is running.
AD_ERROR_STOP_SAMPLING The sampling is stopped. The specified sampling is not available while the sampling has been stopped.
AD_ERROR_START_SAMPLING The sampling is failed. -
AD_ERROR_SAMPLING_TIMEOUT The time-out interval elapsed while the sampling is running. -
AD_ERROR_SAMPLING_FREQ The sampling frequency is invalid.

Specify correct sampling frequency.
AD_ERROR_INVALID_PARAMETER Invalid parameters are specified. Specify correct values.
AD_ERROR_ILLEGAL_PARAMETER The sampling setting is invalid. Invalid sampling conditions are specified.
AD_ERROR_NULL_POINTER A NULL pointer is specified. A NULL pointer is specified for the source data in the data conversion function. The pointer to the buffer that receives the converted data is NULL.
AD_ERROR_GET_DATA The sampling data could not be obtained. Sampling buffer is cleared. The sampling buffer is empty. So no data is returned.
AD_ERROR_USED_DA The DA driver is using the specified function now. Release the functional configuration that the DA driver is using. Then, call the function again.
AD_ERROR_FILE_OPEN Failed to open the file. The specified file does not exist.
AD_ERROR_FILE_CLOSE Failed to close the file. Errors occurred while the file is accessed.
AD_ERROR_FILE_READ Failed to read the file. Errors occurred while the file is accessed. 
AD_ERROR_FILE_WRITE Failed to write the file. Errors occurred while the file is accessed.
AD_ERROR_INVALID_DATA_FORMAT The specified data format is invalid. Use valid data format.
AD_ERROR_INVALID_AVERAGE_OR_SMOOTHING The specified averaging or interpolations are invalid. The number of averaging or interpolations is invalid.
AD_ERROR_INVALID_SOURCE_DATA Data specified as source is invalid. Make sure that specified address of the source data is correct.
AD_ERROR_NOT_ALLOCATE_MEMORY Not enough memory. Not enough memory is available to process.
AD_ERROR_NOT_LOAD_DLL Failed to load DLL. Failed to load DLL. The DLL was not  found.
AD_ERROR_CALL_DLL Failed to call DLL. This function attempts to control the capability that does not support in the DLL.
