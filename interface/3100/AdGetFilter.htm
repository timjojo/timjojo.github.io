AdGetFilter

Description

The AdGetFilter function retrieves the digital filter configuration of the board/card.

Syntax

C

INT AdGetFilter(
    HANDLE hDeviceHandle,
    ULONG *ulFilter
);

Visual Basic

Declare Function AdGetFilter Lib "FbiAd.DLL" ( _
    ByVal hDeviceHandle As Long, _
    ByRef ulFilter As Long _
) As Long

Delphi

function AdGetFilter(
    hDeviceHandle: THandle;
    var ulFilter : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameter

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
  
ulFilter  Points to the variable to obtain digital filter setting.

The digital filter is obtained by the following code.
Code Value Description
AD_DF_8 0 Over sampling rate: 8
AD_DF_16 1 Over sampling rate: 16
AD_DF_32 2 Over sampling rate: 32
AD_DF_64 3 Over sampling rate: 64
AD_DF_128 4 Over sampling rate: 128
AD_DF_256 5 Over sampling rate: 256

Return Value

The AdGetFilter function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

- This function is applicable to the LPC-320724 and PEX-320724.
- The default setting is over sampling rate: 8 (AD_DF_8).

Example

C

ULONG ulFilter;

nRet = AdGetFilter( hDeviceHandle, &ulFilter );

Visual Basic

Dim ulFilter As Long

nRet = AdGetFilter( hDeviceHandle, ulFilter )

Delphi

var
    uFilter: Cardinal;

nRet := AdGetFilter( hDeviceHandle, uFilter );

Retrieves the value of over sampling rate on the board/card specified by hDeviceHandle.
