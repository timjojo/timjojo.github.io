AdSetBoardConfig

Description

The AdSetBoardConfig function configures event handling of the board/card.

Syntax

C 

INT AdSetBoardConfig(
    HANDLE hDeviceHandle,
HANDLE hEvent,
LPADCALLBACK lpCallBackProc,
DWORD dwUser
);

Visual Basic 

Declare Function AdSetBoardConfig "FbiAd.DLL" ( _
   ByVal hDeviceHandle As Long, _
ByVal hEvent   As Long, _
ByVal lpCallBackProc   As Long, _ 
ByVal dwUser As Long _
) As Long 

Delphi

function AdSetBoardConfig(
   hDeviceHandle : THandle;
hEvent : THandle;
lpCallBackProc : FARPROC;
dwUser : DWORD
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle  Specifies the device handle obtained by the AdOpen function.
  
hEvent Specifies an event to be signaled when a sampling terminates or the specified number of samples were successfully acquired. If you don't use event notification, specify it as follows:
  • NULL (C)
  • 0 (Visual Basic)
  • nil (Delphi)

No event notification is available for the default settings.
       
lpCallBackProc Specifies an address of user callback routine to be called when the sampling stops. If you don't use a callback routine, specify it as follows:
  • NULL (C)
  • 0 (Visual Basic)
  • nil (Delphi)

No event notification is available for the default settings.
  
dwUser Specifies a user data to be passed to the user callback routine. If you do not use the user callback routine, please specify 0. The default setting value is 0.

Return Value

The AdSetBoardConfig function returns AD_ERROR_SUCCESS if the process is  successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

The result of this function is updated immediately while a sampling is running. However, this function is applicable only when the AdStartSampling function is performed as an overlapped operation. In Visual Basic Ver.4.0, callback routine is not available.

When you use callback routine in Visual Basic, there are restrictions to use. Refer to "Attentions" for the restrictions.

Examples

C 

hEvent = CreateEvent(0, TRUE, FALSE, NULL);
   
nRet = AdSetBoardConfig( hDeviceHandle, hEvent, CallBackProc, 0 );

Visual Basic

nRet = AdSetBoardConfig( hDeviceHandle, hEvent, _ AddressOf CallBackProc, 0 )

Delphi

nRet := AdSetBoardConfig( hDeviceHandle, hEvent, Addr(CallBackProc), 0 );
Configure event handling on the board/card specified by hDeviceHandle.
