AdSetSamplingConfig

Description

The AdSetSamplingConfig function configures sampling conditions of the board/card.

Syntax

C 

INT AdSetSamplingConfig(
      HANDLE  hDeviceHandle,
PADSMPLREQ  pAdSmplConfig
);

Visual Basic 

Declare Function AdSetSamplingConfig Lib _ "FbiAd.DLL" ( _
   ByVal hDeviceHandle   As Long, _
   ByRef  pAdSmplConfig As ADSMPLREQ _
) As Long         

Delphi

function AdSetSamplingConfig(
hDeviceHandle : THandle;
    var pAdSmplConfig : ADSMPLREQ
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
   
pAdSmplConfig Points to the ADSMPLREQ structure containing sampling conditions.

Return Value

The AdSetSamplingConfig function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • The updated conditions will be valid when the next sampling starts. 

  • This function is not available while a sampling is running on the board/card to which the memory transfer mode is applied.
  • Set the default value to members of AD sampling structure whose setting are not changed. The default setting is obtained by the AdGetSamplingConfig function.  

  • Please refer to "Corresponding Identifier for Each Model," for more details.

Target Products

Refer to "Corresponding Identifiers for Each Model". 

Examples

C 

ADSMPLREQ AdSmplConfig;
    
AdSmplConfig.ulChCount = 4;
AdSmplConfig.SmplChReq[0].ulChNo = 1;
AdSmplConfig.SmplChReq[0].ulRange = AD_5V;
AdSmplConfig.SmplChReq[1].ulChNo = 2;
AdSmplConfig.SmplChReq[1].ulRange = AD_5V;
AdSmplConfig.SmplChReq[2].ulChNo = 3;
AdSmplConfig.SmplChReq[2].ulRange = AD_5V;
AdSmplConfig.SmplChReq[3].ulChNo = 4;
AdSmplConfig.SmplChReq[3].ulRange = AD_5V;
AdSmplConfig.ulSamplingMode = AD_IO_SAMPLING;
AdSmplConfig.ulSingleDiff = AD_INPUT_SINGLE;
AdSmplConfig.ulSmplNum = 1024;
AdSmplConfig.ulSmplEventNum = 0;
AdSmplConfig.fSmplFreq = 1000.0;
AdSmplConfig.ulTrigPoint = AD_TRIG_START;
AdSmplConfig.ulTrigMode = AD_FREERUN;
  
nRet = AdSetSamplingConfig( hDeviceHandle, &AdSmplConfig );
