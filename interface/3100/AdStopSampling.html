AdStopSampling

Description

The AdStopSampling function stops continuous sampling performed as an overlapped operation.

Syntax

C 
INT AdStopSampling(
   HANDLE  hDeviceHandle
);
Visual Basic
Declare Function AdStopSampling Lib "FbiAd.DLL" ( _
   ByVal  hDeviceHandle As Long _

) As Long
Delphi
function AdStopSampling(
hDeviceHandle   : THandle

) : DWORD; stdcall; external 'FbiAd.DLL';

Parameter

hDeviceHandle Specifies the device handle obtained by the AdOpen function.

Return Value

The AdStopSampling function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns an another code. Please refer to error codes.

Comment

If the callback routine is set by using the AdSetBoardConfig function, after this function is called, the callback routine executes.

Examples

C 

nRet = AdStopSampling( hDeviceHandle );
Visual Basic

nRet = AdStopSampling( hDeviceHandle )
Delphi

nRet := AdStopSampling( hDeviceHandle );
Stop the samplings on the board/card specified by hDeviceHandle immediately.
