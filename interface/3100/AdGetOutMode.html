AdGetOutMode

Description

The AdGetOutMode function retrieves the control setting of the external output signals.

Syntax

C 

INT AdGetOutMode(
HANDLE  hDeviceHandle,
ULONG  *ulExTrgMode,
ULONG  *ulExClkMode
);

Visual Basic

Declare Function AdGetOutMode Lib "FbiAd.DLL" (_
ByVal  hDeviceHandle  As Long, _
ByRef  ulExTrgMode As Long, _  
ByRef  ulExClkMode As Long _
) As Long

Delphi

function AdGetOutMode(
hDeviceHandle  : THandle;
var   ulExTrgMode : Cardinal;
var  ulExClkMode : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
             
ulExTrgMode Specifies one of the external trigger output conditions by using the following codes.
Code Value Description
AD_DISABLE 0x80000000 No external trigger output
(default setting value)
AD_TRIG_MODE 2 Trigger timing
AD_BUSY_MODE 3 Sampling status
AD_POST_MODE 4 Post trigger timing
AD_SMP1_MODE 6 Sampling status 1 output 
AD_SMP2_MODE 7 Sampling status 2 output
                          
ulExClkMode Points to a variable to receive the external clock output condition. The external clock output is specified by the following codes.
Code Value Description
AD_DISABLE 0x80000000 The external clock output is disabled.
AD_ENABLE 5 The external clock output is enabled.

Return Value

The AdGetOutMode function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

This function is not available while the sampling is running.

Target Products

Refer to "List of capabilities". 

Examples

C 

ULONG ulExTrgMode, ulExClkMode;
   
nRet = AdGetOutMode( hDeviceHandle, &ulExTrgMode,  &ulExClkMode);

Visual Basic

Dim ulExTrgMode As Long
Dim ulExClkMode As Long
    
nRet = AdGetOutMode( hDeviceHandle, _ ulExTrgMode, ulExClkMode )

Delphi

var  
ulExTrgMode: Cardinal;
ulExClkMode: Cardinal;
    
nRet := AdGetOutMode( hDeviceHandle, ulExTrgMode,  ulExClkMode );
Retrieve the control setting of the external output signals on the board/card specified by hDeviceHandle.
