AdMeasureTemperature

Description

The AdMeasureTemperature function reads temperature of the I/O connector on the board/card. This function is applicable to the PCI-3179 and CTP/CPZ-3179.

Syntax

C

INT AdMeasureTemperature(
HANDLE hDeviceHandle, 
float *fTemperature 
);

Visual Basic

Declare Function AdMeasureTemperature Lib _ "FbiAd.DLL" ( _
   ByVal hDeviceHandle As Long, _
ByRef  Temperature As Single _
) As Long

Delphi

function AdMeasureTemperature(
hDeviceHandle : THandle;
var fTemperature : Single
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
   
fTemperature  Points to a variable to receive the measured temperature.
Note: Centigrade temperature is retrieved as the measured temperature.

Return Value

The AdMeasureTemprature function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

This function can measure temperature at every 0.25 °C (32.45 °F). The temperature range is 0 °C to 50 °C (32 °F to 122 °F).

Examples

C

float fTemperature;
   
nRet = AdMeasureTemperature( hDeviceHadle, &fTemperature );

Visual Basic

Dim fTemperature As Single
  

nRet = AdMeasureTemperature( hDeviceHadle, _ fTemperature )

Delphi

var
   fTemperature: Single;
   
nRet := AdMeasureTemperature( hDeviceHadle, fTemperature );
Read temperature of the I/O connector on the board/card specified by hDeviceHandle.
