ADBOARDSPEC Structure

The ADBOARDSPEC structure contains the specifications of the board/card. This structure is used for the AdGetDeviceInfo function.

- C
 typedef struct {
 ULONG    ulBoardType;
ULONG    ulBoardID;
DWORD    dwSamplingMode;
ULONG    ulChCountS;
ULONG    ulChCountD;
ULONG    ulResolution;
DWORD    dwRange;   
ULONG    ulIsolation;
ULONG    ulDi;
ULONG    ulDo;
} ADBOARDSPEC, *PADBOARDSPEC;

Member
 Description
ulBoardType Receives the board type.
If you use the PCI-3135, this member will contain 3135 in decimal.
  
ulBoardID Receives the board ID (RSW1 value of the board).
If you use the CBI-3133A, CBI-3133B, CBI-3134A, CBI-3134B, this member will contain 0 in decimal.
  
dwSamplingMode Receives the data transfer mode that the board provides.
Bit Data Transfer Mode

0

1

 FIFO

2

3

 Bus master

4 through 31

 Reserved(0)
0:Not supported, 1:Supported
ulChCountS Receives the number of channels for single-ended input.
  
ulChCountD Receives the number of channels for differential input.
  
ulResolution Receives the resolution of the board.
If you use a 12-bit analog-to-digital input board, this member will contain 12.
