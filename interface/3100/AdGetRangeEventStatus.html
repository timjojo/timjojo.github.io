AdGetRangeEventStatus

Description

The AdGetRangeEventStatus function retrieves the status of the full-scale range detection event. 

Syntax

C 

INT AdGetRangeEventStatus(
HANDLE  hDeviceHandle,
ULONG  *ulEventChNo,
ULONG  *ulEventStatus
);

Visual Basic

Declare Function AdGetRangeEventStatus Lib "FbiAd.DLL" (_
ByVal  hDeviceHandle  As Long, _ 
ByRef  ulEventChNo  As Long, _ 
ByRef  ulEventStatus  As Long _
) As Long

Delphi

function AdGetRangeEventStatus(
hDeviceHandle : THandle;
var ulEventChNo  : Cardinal;
var ulEventStatus : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
             
ulEventChNo Points to a variable to receive the channel number where the full-scale range detection event occurred.
                          
ulEventStatus Points to a variable to receive the full-scale range detection status. The variable will contain one of the following codes.
Code Value Description
AD_STATUS_POSITIVE_FULL_SCALE 1 Positive full-scale range
AD_STATUS_NEGATIVE_FULL_SCALE 2 Negative full-scale range

Return Value

The AdGetRangeEventStatus function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • The array for parameters changes in the required size with control mode of the board/card. The board/card of multiplexer mode prepares the array for channels. The board/card of ADC mode parameters the array for the maximum number of channel to use.

Target Products

Refer to "List of Capabilities for Each Model," for more details of the product models and capabilities.

Examples

C 

ULONG ulEventChNo[2];
ULONG ulEventStatus[2];
            
nRet = AdGetRangeEventStatus( hDeviceHandle, ulEventChNo, ulEventStatus );

Visual Basic

Dim ulEventChNo(0 to 1) As Long
Dim ulEventStatus(0 to 1) As Long
          
nRet = AdGetRangeEventStatus( hDeviceHandle, _ ulEventChNo(0), ulEventStatus(0) )

Delphi

var
    ulEventChNo: array[0..1] of Cardinal;
    ulEventStatus : array[0..1] of Cardinal;
                           
nRet := AdGetRangeEventStatus( hDeviceHandle, @ulEventChNo[0], @ulEventStatus[0] );
Retrieves the status of the full-scale range detection event of the board/card specified by hDeviceHandle.
