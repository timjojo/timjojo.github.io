AdLvStartSampling

Description

The AdLvStartSampling function starts continuous sampling on the board/card.

Syntax

C 

INT AdLvStartSampling(
HANDLE  hDeviceHandle,
   ULONG  ulChNo
);

Visual Basic 

Declare Function AdLvStartSampling Lib _ "FbiAd.DLL" ( _
ByVal  hDeviceHandle As Long, _
   ByVal  ulChNo   As Long _
) As Long      

Delphi

function AdLvStartSampling(
hDeviceHandle : THandle;
   ulChNo : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
   
ulChNo Specifies the channel number to acquire samples.

Return Value

The AdLvStartSampling function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • Before using this function, configure the sampling conditions by using the AdLvSetSamplingConfig function.
  • Each channel uses sampling conditions configured by the AdLvSetSamplingConfig function.
  • To terminate the sampling started by this function, use the AdLvStopSampling function instead of the AdStopSampling function.
  • This function is always performed as an overlapped operation.

Examples

C 

nRet = AdLvStartSampling( hDeviceHandle, 1 );

Visual Basic 

nRet = AdLvStartSampling( hDeviceHandle, 1 )

Delphi

nRet := AdLvStartSampling( hDeviceHandle, 1 );
Start the continuous sampling on channel 1 of the board/card specified by hDeviceHandle.
