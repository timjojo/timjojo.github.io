AdBmSetSamplingConfig

Description

The AdBmSetSamlingConfig function configures sampling conditions.

Syntax

C 

INT AdBmSetSamplingConfig(
   HANDLE hDeviceHandle,
PADBMSMPLREQ   pAdBmSmplConfig
);

Visual Basic 

Declare Function AdBmSetSamplingConfig Lib _ "FbiAd.DLL" ( _
   ByVal   hDeviceHandle As Long, _
ByRef pAdBmSmplConfig   As ADBMSMPLREQ _
) As Long                

Delphi

function AdBmSetSamplingConfig(
      hDeviceHandle : THandle;
   var pAdBmSmplConfig : ADBMSMPLREQ
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
   
pAdBmSmplConfig Points to the ADBMSMPLREQ structure containing sampling conditions.

Return Value

The AdBmSetSamplingConfig function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • The updated conditions will be valid when the next sampling starts. 
  • Configure the default setting value for the member of  AD sampling structure  whose setting does not need to be changed. The default values are retrieved by the AdBmGetSamplingConfig function before changing the setting.
  • Refer to the corresponding identifiers for each model for the table of  the corresponding  models and settings. 

Target Products

Refer to "List of Capabilities for Each Model," for more details of the product models and capabilities.

Examples

C 

ADBMSMPLREQ AdBmSmplConfig;
      
AdBmSmplConfig.ulChCount = 4;
AdBmSmplConfig.SmplChReq[0].ulChNo = 1;
AdBmSmplConfig.SmplChReq[0].ulRange = AD_5V;
AdBmSmplConfig.SmplChReq[1].ulChNo = 2;
AdBmSmplConfig.SmplChReq[1].ulRange = AD_5V;
AdBmSmplConfig.SmplChReq[2].ulChNo = 3;
AdBmSmplConfig.SmplChReq[2].ulRange = AD_5V;
AdBmSmplConfig.SmplChReq[3].ulChNo = 4;
AdBmSmplConfig.SmplChReq[3].ulRange = AD_5V;
AdBmSmplConfig.ulSingleDiff = AD_INPUT_SINGLE;
AdBmSmplConfig.ulSmplNum = 1024;
AdBmSmplConfig.ulSmplEventNum = 0;
AdBmSmplConfig.ulSmplRepeat = 1;
AdBmSmplConfig.ulBufferMode = AD_APPEND;
AdBmSmplConfig.fSmplFreq = 1000.0;
AdBmSmplConfig.fScanFreq = 10000.0;
AdBmSmplConfig.ulStartMode = AD_FREERUN;
AdBmSmplConfig.ulStopMode = AD_FREERUN;
    
nRet = AdBmSetSamplingConfig( hDeviceHandle, &AdBmSmplConfig );
