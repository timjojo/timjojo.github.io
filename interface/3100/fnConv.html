fnConv

Description

The fnConv function is a placeholder for a callback routine used in the AdDataConv function. This function is called when each data is converted.

Syntax

C
CONVPROC fnConv(
INT  nCh,   
DWORD  dwCount,   
PVOID lpData   
);
Visual Basic
Sub fnConv(_
ByVal   nCh   As Long,_ 
ByVal   dwCount  As Long,_  
ByRef    lpData   As Any_
)
Delphi
procedure fnConv(
nCh   : Integer;   
dwCount : DWORD;
lpData   : Pointer
); stdcall;

Parameters

nCh Contains the channel number regarding to the data pointed by lpData.
      
dwCount Contains an index of the data.
  
lpData Points to the data to be processed in this function. After processing, restore the processed data into the area pointed by lpData.

Return Value

This function has no return value.
