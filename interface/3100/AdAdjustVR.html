AdAdjustVR

Description

The AdAdjustVR function controls the digitally controlled potentiometer on the board/card for calibration purpose.

Syntax

C 

INT AdAdjustVR(
HANDLE  hDeviceHandle,
ULONG  ulAdjustCh,
ULONG  ulSingleDiff,
ULONG  ulSelVolume,
ULONG  ulControl,
ULONG  ulTap
);

Visual Basic 

Declare Function ADAdjustVR Lib"FbiAd.DLL" (
ByVal  hDeviceHandle As Long, _
ByVal  ulAdjustCh   As Long, _
ByVal ulSingleDiff As Long, _
ByVal  ulSelVolume   As Long, _
ByVal  ulControl   As Long, _
ByVal  ulTap   As Long _

) As Long

Delphi 

function AdAdjustVR(
hDeviceHandle : THandle;
ulAdjustCh  : Cardinal;
ulSingleDiff  : Cardinal;
ulSelVolume : Cardinal;
ulControl : Cardinal;
ulTap   : Cardinal

) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
  
ulAdjustCh Specifies the channel number to adjust. For the board/card whose input channels are multiplexed, please specify 1.
  
ulSingleDiff Specifies the input configuration.
CodeDescription
AD_INPUT_SINGLESingle-ended input
AD_INPUT_DIFFDifferential input
      
ulSelVolume Selects the electronic volume to adjust.
CodeDescription
AD_ADJUST_BIOFFSETBipolar offset
AD_ADJUST_UNIOFFSETUnipolar offset
AD_ADJUST_BIGAINBipolar gain
AD_ADJUST_UNIGAINUnipolar gain
   
ulControl Specifies the operation.
CodeDescription
AD_ADJUST_UPIncreases the volume.
AD_ADJUST_DOWNDecreases the volume.
AD_ADJUST_STORESaves the present value to the non-volatile memory.
AD_ADJUST_STANDBYPlaces the electronic volume device into the standby mode.
AD_ADJUST_NOT_STORENot save the value.
   
ulTap Specifies the number of steps to increase or decrease, in the range of 1 through 100.

Return Value

The AdAdjustVR function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

Some board/card don't support this capability. In such case, this function returns AD_ERROR_NOT_SUPPORTED. You don't have to use this API because the adjustments are done by the AdCalib.exe program included in this product.

Examples

C

nRet = AdAdjustVR( hDeviceHandle, 1, AD_INPUT_SINGLE, AD_ADJUST_BIOFFSET, AD_ADJUST_DOWN, 1 );

Visual Basic

nRet = AdAdjustVR( hDeviceHandle, 1, _ AD_INPUT_SINGLE, AD_ADJUST_BIOFFSET, _ AD_ADJUST_DOWN, 1 )

Delphi

nRet := AdAdjustVR( hDeviceHandle, 1, AD_INPUT_SINGLE, AD_ADJUST_BIOFFSET, AD_ADJUST_DOWN, 1 );
Adjust a bipolar offset on the board/card specified by hDeviceHandle.
