AdFifoGetSamplingData

Description

The AdFifoGetSamplingData function retrieves sampling data and digital data from the board/card. This function is applicable to the PCI-3153 and CTP-3167 with FIFO transfer mode.

Syntax

C 

INT AdFifoGetSamplingData(
   HANDLE hDeviceHandle,
PVOID  pSmplData,
PVOID  pDiData,
ULONG  *ulSmplNum
);

Visual Basic

   ByVal  hDeviceHandle As Long, _
ByRef  pSmplData As Any, _
ByRef  pDiData As Any, _
ByRef  ulSmplNum As Long _
) As Long      

Delphi

function AdFifoGetSamplingData(
   hDeviceHandle  : THandle;
pSmplData : DWORD;
pDiData : Pointer;
var ulSmplNum : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle  Specifies the device handle obtained by the AdOpen function.
          
pSmplData Points to a buffer to receive the sampling data.
                                  
pDiData Points to a buffer to receive the digital data.
                    
ulSmplNum

Points to a variable to receive the number of samples. Before calling this function, specify the number of samples to be received. After the function returned, it receives the number of samples actually stored into the buffer pointed by pSmplData.

Return Value

The AdFifoGetSamplingData function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • This function is available only when the PCI-3153 or CPZ-3167 is sampled using the FIFO mode. If you sample using Programmed I/O mode, the digital data cannot be retrieved.
  • This function transfers samples from the internal buffer of the driver software into user supplied buffer.
  • Data unit of sampling data depends on the resolution of the board/card. Please refer to "Data Format" and the following table.

Examples

C 

PVOID pSmplData;
PVOID pDiData;
HANDLE hHeap;
ULONG ulSmplNum;
ULONG ulSmplChCount;
     
ulSmplNum = 256;
ulSmplChCount = 3;
hHeap = GetProcessHeap();
pSmplData = HeapAlloc( hHeap, HEAP_ZERO_MEMORY, ulSmplNum * ulSmplChCount * sizeof( WORD ) );
pDiData = HeapAlloc( hHeap, HEAP_ZERO_MEMORY, ulSmplNum * ulSmplChCount * sizeof( BYTE ) );
    
nRet = AdFifoGetSamplingData( hDeviceHandle, pSmplData, pDiData, &ulSmplNum);

Visual Basic

Dim ulSmplNum As Long
     
ulSmplNum = 256

Delphi

var
pSmplData: Pointer;
pDiData: Pointer;

ulSmplChCount: Cardianl;
       
ulSmplNum := 256;
ulSmplChCount := 3;
pSmplData := AllocMem(ulSmplNum * ulSmplChCount * SizeOf(Word));
pDiData := AllocMem(ulSmplNum * ulSmplChCount * SizeOf(Byte));
     
nRet := AdFifoGetSamplingData( hDeviceHandle, pSmplData, pDiData, ulSmplNum );
Retrieve 256 samples (2 byte/sample) per channel for 3 channels on the 12-bit resolution board/card specified by hDeviceHandle. Samples are stored into the buffer pointed by pSmplData. The digital data is stored into the buffer pointed by pDiData. The ulSmplNum parameter contains the number of samples that are actually acquired after the sampling is completed.
