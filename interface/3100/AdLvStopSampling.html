AdLvStopSampling

Description

The AdLvStopSampling function stops continuous sampling performed as an overlapped operation.

Syntax

C
INT AdLvStopSampling(
HANDLE hDeviceHandle,
   ULONG ulChNo
);
Visual Basic
Declare Function AdLvStopSampling Lib "FbiAd.DLL" (_
ByVal hDeviceHandle As Long,_
   ByVal ulChNo   As Long_

) As Long
Delphi
function AdLvStopSampling(
hDeviceHandle : THandle;
ulChNo : DWORD
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
  
ulChNo Specifies the channel number to be stopped the sampling.

Return Value

The AdLvStopSampling function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns an another code. Please refer to error codes.

Comments

If the event notification and the callback routine are set by using the AdLvSetSamplingConfig function, after this function is called, the event is signaled and the callback function is executed.
     
This function cannot stop the sampling started by the AdStartSampling function. 

Examples

C 

nRet = AdLvStopSampling( hDeviceHandle, 1 );
Visual Basic

nRet = AdLvStopSampling( hDeviceHandle, 1 )
Delphi

nRet := AdLvStopSampling( hDeviceHandle, 1 );
Stop the samplings on the board/card specified by hDeviceHandle immediately.
