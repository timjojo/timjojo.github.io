AdReadAdjustVR

Description

The AdReadAdjustVR function loads the electric volume value. When you use the board/card that does not support this function, AD_ERROR_NOT_SUPPORTED is returned.

Syntax

C 

INT AdAReaddjustVR(
   HANDLE  hDeviceHandle,
ULONG ulAdjustCh
);

Visual Basic 

Declare Function AdReadAdjustVR "FbiAd.DLL" ( _
ByVal hDeviceHandl As Long, _
   ByRef  ulAdjustCh As Long _
) As Long       

Delphi

function AdGetSamplingConfig(
hDeviceHandle : THandle;
  ulAdjustCh : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
   
ulAdjustCh Specifies the channel number to calibrate. Specifies 1 for the multiplexer board/card. For multiplexer boards/cards, specifies 1.

Return Value

The AdReadAdjustVR function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • When the Cancel button is clicked after calibration completion in the calibration program, the value in EEPROM is loaded.
