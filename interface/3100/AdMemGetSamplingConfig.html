AdMemGetSamplingConfig

Description

The AdMemGetSamplingConfig function retrieves the configured sampling conditions. 

Syntax

C 
INT AdMemGetSamplingConfig(
HANDLE  hDeviceHandle,
PADMEMSMPLREQ pAdMemSmplConfig
);
Visual Basic
Declare Function AdMemGetSamplingConfig Lib _ "FbiAd.DLL" ( _
ByVal hDeviceHandle As Long, _
ByRef pAdMemSmplConfig As ADMEMSMPLREQ _
) As Long
Delphi
function AdMemGetSamplingConfig(
hDeviceHandle   : THandle;
var pAdMemSmplConfig : ADMEMSMPLREQ
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
    
pAdMemSmplConfig Points to the ADMEMSMPLREQ structure to store the current setting sampling condition.

Return Value

The AdMemGetSamplingConfig function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

When executing this function as soon as the device is opened, you can retrieve the default setting value of the ADMEMSMPLREQ structure. 
This function is applicable to the CSI/CBI-320110, CBI-320110TK, CBI-320110TL, CTP/CPZ-360810, LPC-320910 and PEX-320910.

Examples

C 
ADBMSMPLREQ AdBmSmplConfig;
nRet = AdBmGetSamplingConfig( hDeviceHandle, &AdBmSmplConfig );
Visual Basic

Dim AdBmSmplConfig As ADBMSMPLREQ
nRet = AdBmGetSamplingConfig( hDeviceHandle, _ AdBmSmplConfig ) 
Delphi

var
   AdBmSmplConfig: ADBMSMPLREQ;
nRet := AdBmGetSamplingConfig( hDeviceHandle, AdBmSmplConfig );
Retrieve the sampling conditions on the board/card specified by hDeviceHandle.
