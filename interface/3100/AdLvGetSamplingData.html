AdLvGetSamplingData

Description

The AdLvGetSamplingData function retrieves sampling data from the board/card.

Syntax

C 

INT AdLvGetSamplingData(
      HANDLE hDeviceHandle,
ULONG  ulChNo,
PVOID   pSmplData,
ULONG  *ulpSmplNum
);

Visual Basic 

Declare Function AdLvGetSamplingData Lib "FbiAd.DLL" (_
   ByVal  hDeviceHandle As Long, _
ByVal  ulChNo   As Long, _
ByRef  pSmplData   As Any, _
ByRef  ulpSmplNum   As Long _
) As Long      

Delphi

function AdLvGetSamplingData(
   hDeviceHandle : THandle;
ulChNo : DWORD;
pSmplData   : Pointer;
var ulpSmplNum : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle  Specifies the device handle obtained by the AdOpen function.
                       
ulChNo Specifies the channel number to acquire samples.
                             
pSmplData Points to a buffer to receive samples.
                             
ulpSmplNum  Points to a variable to receive the number of samples. Before calling this function, specify the number of samples to be received. After the function returned, it receives the number of samples actually stored into the buffer pointed by pSmplData.

Return Value

The AdLvGetSamplingData function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • This function transfers samples from the internal buffer of the driver software into user supplied buffer.
  • Data unit depends on the resolution of the board/card. Please refer to "Data Format" and the following table.

Examples

C 

PVOID pSmplData;
HANDLE hHeap;
ULONG ulSmplNum
  
ulSmplNum = 256;
hHeap = GetProcessHeap();
pSmplData = HeapAlloc( hHeap, HEAP_ZERO_MEMORY, ulSmplNum * sizeof( WORD ) );
  
nRet = AdLvGetSamplingData( hDeviceHandle, 1, pSmplData, &ulSmplNum);

Visual Basic 

Dim SmplData(255) As Integer
Dim ulSmplNum As Long
  
ulSmplNum = 256
  

nRet = AdLvGetSamplingData( hDeviceHandle, 1, _ SmplData(0), ulSmplNum )

Delphi

var
pSmplData: Pointer;
ulSmplNum: Cardinal;
    
pSmplData := AllocMem(256 * SizeOf(Word));
     
ulSmplNum := 256;
nRet := AdLvGetSamplingData( hDeviceHandle, 1, pSmplData, ulSmplNum );
Retrieve 256 samples from channel 1 on the board/card specified by hDeviceHandle. Samples are stored into the buffer pointed by pSmplData. The ulSmplNum parameter contains the number of samples that are actually acquired after the sampling is completed.
