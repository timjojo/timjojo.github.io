AdMemGetDiPattern

Description

The AdMemGetDiPattern function retrieves the analog-to-digital input pattern matching conditions. 

Syntax

C 
INT AdMemGetDiPattern(
HANDLE  hDeviceHandle,
ULONG *bPatternTrug
);
Visual Basic
Declare Function AdMemGetDiPattern Lib _ "FbiAd.DLL" ( _
ByVal hDeviceHandle As Long, _
ByRef ulPatternTrug As Long _
) As Long
Delphi
function AdMemGetDiPattern(
hDeviceHandle   : THandle;
var ulPatternTrug : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
    
ulPatternTrig Points to a variable to retrieve the condition setting of pattern matching trigger setting.
Code Description
AD_DISABLE No setting (default setting)
AD_LOW_LEVEL Input signal status: 0 (low level)
AD_HIGH_LEVEL Input signal status: 1 (high level)
AD_DOWN_EDGE  Falling edge
AD_UP_EDGE  Rising edge

Return Value

The AdMemGetDiPattern function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • This function is not available while the sampling is running.
  • This function is enable to use when the sampling end condition is configured to the DI pattern matching （AD_STOP_DI_EQ）by using the AdMemSetSamplingConfig function.
  • Please prepare the array for the maximum number of DI channels.
  • This function is applicable to the CSI/CBI-320110, CBI-320110TK, CBI-320110TL, LPC-320910, PEX-320910 and CTP/CPZ-360810.

Examples

C 
ULONG ulPatternTrig[8];
   
nRet = AdMemGetDiPattern( hDeviceHandle ulPatternTrig );
Visual Basic

Dim ulPatternTrig(7) As Long
   
nRet = AdMemGetDiPattern( hDeviceHandle, _ ulPatternTrig(0) )
Delphi

var
   ulPatternTrig: array[0..7] of Cardinal;
    
nRet := AdMemGetDiPattern( hDeviceHandle, @ulPatternTrig[0] );
Retrieve the analog-to-digital input pattern matching conditions on the board/card specified by hDeviceHandle.
