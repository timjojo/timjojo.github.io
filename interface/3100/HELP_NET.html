15. How to Use .NET

15.1 Overview

This chapter explains how to use the GPF-3100 for .NET. 
The following table shows the supported languages.
Supported Languages Microsoft Visual C++ .NET 2002, .NET 2003, 2005, 2008, 2010
Microsoft Visual C# .NET 2002, .NET 2003, 2005, 2008, 2010
Microsoft Visual Basic .NET 2002, .NET 2003, 2005, 2008, 2010
Borland Delphi 8 for the Microsoft .NET Framework
Before building a program, convert *.csproj file for Visual C# 2005 and *.vbproj file for Visual Basic 2005 by the Visual Studio conversion wizard.

15.2 Class Library

15.2.1 Creating the Class Library

To call DLL functions on .NET, you need to prepare class library first.

Visual C# .NET 

1. Start Visual Studio.
2. Open the following project file.
<installation destination> 
\interface\GPF3100\samples\CS_NET\IFCAD\IFCAD.csproj
After building the project file, IFCAD.DLL is created in the bin folder. 

Visual Basic .NET

1. Start Visual Studio.
2. Open the following project file.
<installation destination>
\interface\GPF3100\samples\VB_NET\IFCAD\IFCAD.vbproj 
After building the project file, IFCAD.DLL is created in the bin folder. 

Delphi 8

1. Start Delphi.
2. Open the following project file.
<installation destination>
\interface\GPF3100\samples\DELPHI_NET\IFCAD\IFCAD.bdsproj
After building the project file, IFCAD.DLL is created in the bin folder.

 

15.2.2 Adding Class Library Reference

Visual C# .NET and Visual Basic .NET

1. Start Visual Studio.
2. Select Project > Add Reference.
3. Click the Browse button.
4. Specify the class library DLL to refer.
5. The DLL file will be displayed in Selected Components.
6. Click OK.
7. Add the namespace of InterfaceCorpDllWrap as below at the header of your source file to call the DLL functions.

Delphi 8

1. Start Delphi 8.
 
2. Select Project > Add Reference.
 
3. Specify the class library DLL to refer.
 
4. The DLL file will be displayed.
 
5. Click OK.
   
6. Add the namespace of InterfaceCorpDllWrap as below at the header of your source file to call the 
DLL functions.
Visual C# .NET
using InterfaceCorpDllWrap;
Visual Basic .NET
Imports InterfaceCorpDllWrap
Delphi 8
uses InterfaceCorpDllWrap;
uses InterfaceCorpDllWrap.IFCAD; (for callback function)

16. Sample Programs for .NET

This chapter describes the outline of sample programs prepared for each .NET language.

16.1 Execution Procedure

The GPF-3100 has sample programs for Visual C# .NET, Visual Basic .NET., and Delphi 8. For Visual C++ .NET, use sample programs for Visual C++.
Each sample programs are created by Visual Studio .NET2003.

Execution procedure

Executable files of the sample programs are not included with this product. 
Compile the source code and create executable file to start.

Visual C# .NET

1. Start Visual Studio.
 
2. Select File > Open > Project.
 
3. Open the project file, *.csproj.
 
4. Build the project file.
 
5. Run the executable file, *.exe.

Visual C# .NET

1. Start Visual Studio.
 
2. Select File > Open > Project.
 
3. Open the project file, *.vbproj.
 
4. Build the project file.
 
5. Run the executable file, *.exe.

Visual Basic .NET

1. Start Visual Studio.
 
2. Select File > Open > Project.
 
3. Open the project file, *.csproj.
 
4. Build the project file.
 
5. Run the executable file, *.exe.

Delphi 8

1. Start Delphi.
 
2. Select File > Open Project.
 
3. Open the project file, *.dpr.
 
4. Build the project file.
 
5. Run the executable file, *.exe.


Note : The class library will be automatically built at the same time as opening the
project files.
