Sampling Clock Error
A sampling clock error occurs when the next sampling clock signal inputs during channel scanning. If the sampling clock error is detected, the board continues or terminates the sampling depending on the setting.
You can select sampling stop/continue when a sampling clock error occurs by the AdBmSetSamplingConfig function. If sampling continue is selected, error clock is ignored and sampling is continued.
