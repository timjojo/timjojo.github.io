Event Generation Using the FIFO Mode
When only one channel is sampled, the event can be occurred whenever s achieves at the number of specified samples in the range from 1 to 2048. When sampling two or more channels, an event can be occurred whenever the s/n data
     (1 <= s/n <= 2048/n) per channel.

     s: the number of acquired samples
     n: the number of sampling channels

Sampling data is stored into sampling buffer every event generation timing.
