Event Notification for the Sampling Counter Event
When the number of samples is greater than 524288, events can be notified whenever the number of the sampled data reaches multiples of predefined event intervals based on the number of samples of 1024, 2048, 4096, 8192, or 16384. While the number of samples is less than or equal to 5242488, the event notification is not available.

The number of the sampled data is updated at the event cycle. When the number of the sampled data is specified 0, 16384 is applied to the predefined event interval.
