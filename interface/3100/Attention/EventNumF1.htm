Event Generation Using the FIFO Mode
When you acquire the data using the FIFO mode, events occur whenever the number of sampling data reaches multiplies of the following value depending on the number of channels used.
Sampling data is stored into a sampling buffer in units of the following numbers.

The configurable event number differs depending on the sample frequency obtained by the following equation. 

SmplFreq = sampling frequency * the number of channels
