Number of Samples that are Sampled Before or After the Trigger
When you acquire the data by using the FIFO mode, the number of post-trigger samples for stop-trigger can be specified from 2 through 65537. For I/O transfer  mode, the number of post-trigger samples for stop-trigger can be specified from 1 through 1073741824.
