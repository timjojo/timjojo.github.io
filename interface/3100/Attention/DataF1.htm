- Analog-to-Digital Input Data (PCI-3153)
When you use the PCI-3513 with FIFO mode in the GPC-3100, version 1.41-26 or older versions, the information of input and FIFO status are added as the table below. 

However, the later version does not add the information. Use the AdFifoGetSamplingData function to get information of input and FIFO status.

Analog Data

-PCI-3153

bit15 bit14 bit13 bit12 bit11 bit10 bit9 bit8 bit7 bit6 bit5 bit4 bit3 bit2 bit1 bit0
FE IN2 IN1 0 12-bit analog data

FE:FIFO status

When the bit is 1, the FIFO is empty. When the bit is 0, the FIFO is not empty. 
When FIFO is empty, the 12-bit analog data is 0. 

IN1, IN2:input IN1, IN2
