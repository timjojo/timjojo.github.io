Functional Differences in Revision ID
CBI-3133A, CBI-3133B, CBI-3134A, CBI-3134B
- RevisionID = 1
Function Description
Full-scale range detection  Not supported.
Programmed I/O mode Supported
FIFO mode Not supported.
- RevisionID = 2
Function Description
Full-scale range detection Not supported.
Programmed I/O mode Supported
FIFO mode Fixed number sampling and infinite sampling are supported.
Trigger capabilities and trigger delay capabilities are not supported.
- RevisionID = 3 or later
Function Description
Full-scale range detection Supported
Programmed I/O mode Supported
FIFO mode Supported

The AdBmSetSamplingConfig, AdBmGetSamplingConfig, AdSetRangeEvent, AdGetRangeEventStatus, and AdResetRangeEvent functions are available for the boards of Revision ID 3.
Note: You can check the Revision ID by using the AdCommonGetPciDeviceInfo function.
