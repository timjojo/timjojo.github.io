Using the Provided Functions
Include the header files of dynamic link library in your C, Visual Basic, or Delphi program source or add them to the project files. Build your program with the import library file. Refer to the syntax described in the Help. Also refer to  the FBIAD.H file for C, the FBIAD.BAS file for Visual Basic, and the FBIAD.PAS file for Delphi.
