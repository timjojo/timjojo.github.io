Import Library File
The import library file, FBIAD.LIB or FBIADDC.LIB, included in this product is meant to be used with Microsoft Visual C++.
If you want to use this product with Borland C++ or Borland C++Builder, you need to make an import library file from the dynamic link library file, FBIAD.DLL or FBIADDC.DLL, by using the Borland's IMPLIB utility. 
If you have any questions about the use of IMPLIB or other such problems, please refer to the user guide of Borland C++ or C++Builder.
