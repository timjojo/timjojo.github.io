External Clock, External Trigger
Programmed I/O mode 
A signal that is connected to the EXINT IN pin is handled as an external clock signal or an external trigger signal depending on the settings.
Therefore, an external clock and external trigger cannot be used simultaneously. 

FIFO mode
 -A signal that is connected to the EXCLK IN pin is handled as an external clock signal.
 -A signal that is connected to the EXTRG IN pin is handled as an external trigger signal.

Please refer to "External Trigger Function."
