AdInputDI

Description

The AdInputDI function reads data from the digital input pins on the board/card.

Syntax

C

INT AdInputDI(
HANDLE  hDeviceHandle,
   DWORD  *dwData
);

Visual Basic

Declare Function AdInputDI Lib "FbiAd.DLL" ( _
ByVal  hDeviceHandle As Long, _
   ByRef  dwData  As Long _

) As Long

Delphi

function AdInputDI(
hDeviceHandle : THandle;
var dwData   : DWORD

) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
   
dwData Points to a variable to receive the digital input data. Please refer to "Data Format."

Return Value

The AdInputDI function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment


The AdInputDI function is not applicable to the boards/cards (PCI-3161 and PCI-3163) that have no digital input pins.

To use digital input on the PCI-3525, and CTP/CPZ-3525, change either CN3 or CN4 to digital input pin by the AdSetFunction function.

To use digital input on the PCI-361812, change the pin to be used to digital input pin by the AdSetFunction function.

Examples

C

DWORD dwData;
nRet = AdInputDI( hDeviceHandle, &dwData );

Visual Basic

Dim dwData As Long
nRet = AdInputDI( hDeviceHandle, dwData )

Delphi

var
    dwData: DWORD;
nRet := AdInputDI( hDeviceHandle, dwData );
Read the status of general purpose digital input pins on the board/card specified by hDeviceHandle.
