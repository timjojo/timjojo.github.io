AdResetRangeEvent

Description

The AdResetRangeEvent function resets the full-scale range detection event.

Syntax

C 

INT AdResetRangeEvent(
HANDLE  hDeviceHandle
);

Visual Basic

Declare Function AdResetRangeEvent Lib _ "FbiAd.DLL" ( _
ByVal  hDeviceHandle  As Long _
) As Long

Delphi

function AdResetRangeEvent(
  hDeviceHandle : THandle
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameter

hDeviceHandle Specifies the device handle obtained by the AdOpen function.

Return Value

The AdResetRangeEvent function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • After the full-scale range detection event occurred, the next event is not occurred until the existing event is reset by using this function. 
  • Refer to "List of Capabilities for Each Model," for more details of the product models and capabilities.

Examples

C 

nRet = AdResetRangeEvent( hDeviceHandle );

Visual Basic

nRet = AdResetRangeEvent( hDeviceHandle )

Delphi

nRet := AdResetRangeEvent( hDeviceHandle );
Reset the full-scale detection on the board/card specified by hDeviceHandle.
