AdReadFile

Description

The AdReadFile function reads the sampling data from a file. The binary and CSV formats are supported.

Syntax

C 

INT AdReadFile(
LPCTSTR  pszPathName,
PVOID  pSmplData,
ULONG  ulFormCode
);
Visual Basic
Declare Function AdReadFile Lib _ "FbiAdDC.DLL" ( _
ByVal  pszPathName As String, _
ByRef  pSmplData As Any, _
ByVal  ulFormCode As Long _
) As Long
Delphi
function AdReadFile(
pszPathName : Pointer;
pSmplData : Pointer;
ulFormCode : Cardinal

) : DWORD; stdcall; external 'FbiAdDC.DLL';

Parameters

pszPathName Specifies the path to the data file which the sampling data is stored.
        
pSmplData Points to the buffer to store the sampling data from the file. The caller software previously should allocate the buffer enough to store the every data from the file.
Data Form Buffer Size
Physical values The number of data * 16 bits
(the physical value data is treated as float format. )
Binary (8 bits through 24 bits) File size to be read
           
ulFormCode Specifies the sampling data format.  
CodeDescription
AD_DATA_PHYSICALPhysical value (voltage [V] or current [mA] ) in the CSV format.
AD_DATA_BIN88-bit binary (file: binary format)
AD_DATA_BIN1212-bit binary (file: binary format)
AD_DATA_BIN1616-bit binary (file: binary format)
AD_DATA_BIN2424-bit binary (file: binary format)

Return Value

The AdReadFile function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

The data format is not changed after reading from the file. Commas are removed while reading from the CSV format file.

Examples

C

char *pszPathName = "DATA.CSV";
nRet = AdReadFile( pszPathName, pSmplData, AD_DATA_PHYSICAL );

Visual Basic

Dim pszPathName As String
pszPathName = "DATA.CSV" & Chr(0)
nRet = AdReadFile( pszPathName, pSmplData, _ AD_DATA_PHYSICAL )

Delphi

var
   pszPathName: array[0..15] of Char;
       
StrCopy(pszPathName, 'DATA.CSV');
nRet := AdReadFile( pszPathName, pSmplData, AD_DATA_PHYSICAL );
Read samples from the DATA CSV file in CSV data format into the pSmplData array.
