AdReadSamplingBuffer

Description

The AdReadSamplingBuffer function reads sampling data contained in the sampling buffer.

Syntax

C 

INT AdReadSamplingBuffer(
   HANDLE hDeviceHandle,
LONG  lOffset,
ULONG  *ulSmplNum,
PVOID  pSmplData
);

Visual Basic 

Declare Function AdReadSamplingBuffer Lib _ "FbiAd.DLL" ( _
   ByVal  hDeviceHandle As Long, _
ByVal  lOffset  As Long, _
ByRef  ulSmplNum  As Long, _
ByRef  pSmplData  As Any _
) As Long      

Delphi

function AdReadSamplingBuffer(
   hDeviceHandle : THandle;
lOffset : Cardinal;
var ulSmplNum  : Pointer;
pSmplData  : Cardinal
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle  Specifies the device handle obtained by the AdOpen function.
        
lOffset Specifies the index of the sampling buffer to start reading the sampling data. This index is specified per number of sample. When -1 is specified, this function reads the latest data of ulSmplNum.
 
ulSmplNum Points to a variable to receive the number of samples. Before calling this function, specify the number of samples to be received. After the function returned, it receives the number of samples actually stored into the buffer pointed by pSmplData.
   
pSmplData Points to a buffer to receive samples.

Return Value

The AdReadSamplingBuffer function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comments

  • This function can retrieve the data from specified address in the sampling buffer. The retrieved data cannot be cleared from the sampling buffer.
  • This function can retrieve the sampling data in the sampling buffer even if the AdGetSamplingData function is in progress. This function has no effect to the AdGetSamplingData function.

Examples

C 

PVOID pSmplData;
HANDLE hHeap;
ULONG ulSmplNum;
ULONG ulSmplChCount;
     
ulSmplNum = 256;
ulSmplChCount = 3;
hHeap = GetProcessHeap();
pSmplData = HeapAlloc( hHeap, HEAP_ZERO_MEMORY, ulSmplNum * ulSmplChCount * sizeof( WORD ) );
     
nRet = AdReadSamplingBuffer( hDeviceHandle, 100, &ulSmplNum, pSmplData);

Visual Basic 

Dim SmplData(2,255) As Integer
Dim ulSmplNum As Long
   
ulSmplNum = 256
   
nRet = AdReadSamplingBuffer( hDeviceHandle, _ 100, ulSmplNum, SmplData(0,0) )

Retrieve 101 to 256 samples contained in the sampling buffer on the 12-bit or 16-bit resolution board/card specified by hDeviceHandle. The ulSmplNum parameter contains the number of samples that are actually acquired after the sampling is completed.
