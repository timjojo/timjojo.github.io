AdGetDeviceInfo

Description

The AdGetDeviceInfo function retrieves specifications of the board/card.

Syntax

C 

INT AdGetDeviceInfo(
   HANDLE hDeviceHandle,
   PADBOARDSPEC   pAdBoardSpec
);

Visual Basic 

Declare Function AdGetDeviceInfo Lib _ "FbiAd.DLL" ( _
   ByVal   hDeviceHandle  As Long, _
   ByRef pAdBoardSpec As ADBOARDSPEC _
) As Long 

Delphi

function AdGetDeviceInfo(
   hDeviceHandle : THandle;
  var pAdBoardSpec : ADBOARDSPEC  
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameters

hDeviceHandle Specifies the device handle obtained by the AdOpen function.
        
pAdBoardSpec Points to the ADBOARDSPEC structure to receive specifications of the board/card.

Return Value

The AdGetDeviceInfo function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Examples

C 

ADBOARDSPEC AdBoardSpec;
nRet = AdGetDeviceInfo( hDeviceHandle, &AdBoardSpec );

Visual Basic 

Dim AdBoardSpec As ADBOARDSPEC

nRet = AdGetDeviceInfo( hDeviceHandle, _ AdBoardSpec )

Delphi

var AdBoardSpec: ADBOARDSPEC;
nRet = AdGetDeviceInfo( hDeviceHandle, AdBoardSpec );
Retrieve specifications of the board/card specified by hDeviceHandle.
