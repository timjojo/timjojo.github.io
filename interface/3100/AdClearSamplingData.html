AdClearSamplingData

Description

The AdClearSamplingData function clears the sampling data in the sampling buffer.

Syntax

C 

INT AdClearSamplingData(
   HANDLE  hDeviceHandle
);

Visual Basic 

Declare Function AdClearSamplingData Lib _ "FbiAd.DLL" ( _
   ByVal hDeviceHandle As Long _
) As Long         

Delphi

function AdClearSamplingData(
   hDeviceHandle : THandle
) : DWORD; stdcall; external 'FbiAd.DLL';

Parameter

hDeviceHandle Specifies the device handle obtained by the AdOpen function.

Return Value

The AdClearSamplingData function returns AD_ERROR_SUCCESS if the process is successfully completed. Otherwise, this function returns another code. Please refer to the error codes.

Comment

This function is not available while the sampling is running.
This function automatically clears the buffer at the beginning of the sampling regardless of this function. 

Examples

C 

nRet = AdClearSamplingData( hDeviceHandle );

Visual Basic 

nRet = AdClearSamplingData( hDeviceHandle )

Delphi

nRet := AdClearSamplingData( hDeviceHandle );
Clear the sampling data in the buffer on the board/card specified by hDeviceHandle.
